## [gauge[type=bullet]](./gauge/bullet)
bullet 게이지 모델.<br/>
현재 값을 목표 값과 비교해서 표시한다.
또, [band](#band) 를 같이 표시하면, 여러 값 범위 중 어디에 속한 상태인 지를 나타낼 수 있다.
## [gauge[type=circle]](./gauge/circle)
원형 게이지 모델.<br/>
지정된 최대 각도 내에서 현재 값의 비율을 원호로 표시한다.
## [gauge[type=clock]](./gauge/clock)
시계 게이지 모델.<br/>
원형 아날로그 시계를 표시한다.
## [gauge[type=linear]](./gauge/linear)
선형 게이지 모델.<br/>
현재 값을 선형 막대로 표시한다.
또, [band](#band) 를 같이 표시하면, 여러 값 범위 중 어디에 속한 상태인 지를 나타낼 수 있다.
