## grid
visible 기본값이 undefined이다.
visible이 undefined나 null로 지정되면, 축 위치에 따라 visible 여부가 결정된다.
## Properties

### circular: `boolean{:javascript}`[#circular]
polar 차트일 때 그리드 선을 원형으로 그릴 지 여부.  
`default: true`  

### endVisible: `boolean{:javascript}`[#endVisible]
끝 값에 표시되는 그리드 선을 표시할 지 여부.  
`default: true`  

### startVisible: `boolean{:javascript}`[#startVisible]
시작 값에 표시되는 그리드 선을 표시할 지 여부.  
`default: false`  

### style: `SVGStyles | string{:javascript}`[#style]
스타일셋 혹은 class selector.  

### visible: `boolean{:javascript}`[#visible]
표시 여부.\
모델에 따라 기본값이 다르다.
