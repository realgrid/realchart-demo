## [title](/config/config/title)

## Properties

### align[#align]
축 내에서 타이틀의 위치.\
 **[offset](#offset)** 으로 경계 지점과의 간격을 지정할 수 있다.  


default: <span style={{ color: 'var(--shiki-color-text)', fontWeight: 700 }}>AxisTitleAlign.MIDDLE</span>
### backgroundStyle: `SVGStyles | string{:js}`[#backgroundStyle]
타이틀 배경 스타일.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fillOpacity | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeDasharray | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |
### gap: `number{:js}`[#gap]
타이틀과 틱 label들, 혹은 축 선 사이의 간격을 픽셀 단위로 지정한다.  
default: <span style={{ color: 'var(--shiki-color-text)', fontWeight: 700 }}>5</span> pixels
### offset: `number{:js}`[#offset]
**[align](#align)**  기준으로 배치할 때 경계 지점과의 간격.\
plus 값을 지정하면 차트 안쪽으로 들어온다.
align이 'middle'일 때는 음수일 때 축 시작값 쪽으로 밀린다.  
default: <span style={{ color: 'var(--shiki-color-text)', fontWeight: 700 }}>0</span>
### rotation: `0 | 90 | 270 | -90 | -270{:js}`[#rotation]
텍스트 회전 각도.\
수직 축일 때는 0, 90, 270, -90, -270 중 하나로 지정할 수 있다.
수평 축일 때는 무조건 0이다.  
default: <span style={{ color: 'var(--shiki-token-keyword)', fontWeight: 700 }}>undefined</span> 수직 축일 때 왼쪽에 표시되면 90, 오른쪽에 표시되면 270, 수평 축일 때는 0
### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fillOpacity | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeDasharray | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |
### text: `string{:js}`[#text]
타이틀 텍스트.\
지정하지 않으면 타이틀 영역 자체가 없어진다.  

### visible: `boolean{:js}`[#visible]
표시 여부.  
default: <span style={{ color: 'var(--shiki-token-keyword)', fontWeight: 700 }}>true</span>
