## Properties

### align[#align]
축 내에서 타이틀의 위치.\
 offset 으로 경계 지점과의 간격을 지정할 수 있다.  


`default: AxisTitleAlign.MIDDLE`  

### backgroundStyle: `SVGStyles | string{:javascript}`[#backgroundStyle]
타이틀 배경 스타일.  

### gap: `number{:javascript}`[#gap]
타이틀과 틱 label들, 혹은 축 선 사이의 간격을 픽셀 단위로 지정한다.  
`default: 5` pixels 

### offset: `number{:javascript}`[#offset]
align  기준으로 배치할 때 경계 지점과의 간격.\
plus 값을 지정하면 차트 안쪽으로 들어온다.
align이 'middle'일 때는 음수일 때 축 시작값 쪽으로 밀린다.  
`default: 0`  

### style: `SVGStyles | string{:javascript}`[#style]
스타일셋 혹은 class selector.  

### text: `string{:javascript}`[#text]
타이틀 텍스트.\
지정하지 않으면 타이틀 영역 자체가 없어진다.  

### rotatin: ` |  |  |  | {:javascript}`[#rotatin]
텍스트 회전 각도.\
수직 축일 때는 0, 90, 270, -90, -270 중 하나로 지정할 수 있다.
수평 축일 때는 무조건 0이다.  
`default: undefined` 수직 축일 때 왼쪽에 표시되면 90, 오른쪽에 표시되면 270, 수평 축일 때는 0 

### visible: `boolean{:javascript}`[#visible]
표시 여부.\
모델에 따라 기본값이 다르다.
