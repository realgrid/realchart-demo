## **label**
축 label 설정 모델.<br/>
## Properties

### autoArrange: `'none'|'rotate'|'rows'|'step'{:js}`[#autoArrange]
기본 설정이나 [step](#step) , [rows](#rows) , [rotation](#rotation) 속성들을 명시적으로 설정한 경우에도
label들이 본래 차지하는 공간을 초과할 때,
label들을 재배치하는 방식을 지정한다.  
- `'none'` label들이 겹치도록 놔둔다.<br/>
- `'rotate'` -45도 회전시킨다.
- `'rows'` label들이 겹치지 않도록 여러 줄로 나누어 배치한다.
- `'step'` label들이 겹치지 않도록 건너 뛰면서 배치한다.<br/> [startStep](#startStep) 으로 지정된 step부터 배치된다.

<DefaultValue value="'rotate'" contents=""></DefaultValue>
### autoContrast: `boolean{:js}`[#autoContrast]
텍스트가 data point 내부에 표시되는 경우 포인트 색상과 대조되도록 표시한다.


밝게 표시할 때는 [lightStyle](#lightstyle) 을 적용하고,
어둡게 표시할 때는 [darkStyle](#darkstyle) 이 적용된다.
brightStyle이 지정되지 않으면 <Content type="expression" value="'rct-text-light'"></Content>이,
darkStyle이 지정되지 않으면 <Content type="expression" value="'rct-text-dark'"></Content>가 기본 적용된다.  
<DefaultValue value="true" contents=""></DefaultValue>

### effect: `'background'|'none'|'outline'{:js}`[#effect]
- `'background'` 텍스트 배경 상자를 표시한다.      배경 상자에 [backgroundStyle](#backgroundStyle) 이 적용된다.  스타일이 적용되지 않으면 기본 <Content type="expression" value="'rct-text-background'"></Content>이 적용된다.
- `'none'` 효과 없음.
- `'outline'` 텍스트 색상과 대조되는 색상으로 텍스트 외곽을 구분 표시한다.

<DefaultValue value="'none'" contents=""></DefaultValue>
### firstOverflow: `'fit'|'hidden'|'pull'{:js}`[#firstOverflow]
첫번째 tick 라벨이 차트나 분할 경계를 넘어갈 때 표시 방식.<br/>  
- `'fit'` 축 경계를 넘어선 만큼 끌어당겨서 표시한다.<br/>  실제 표시 위치와 달라지므로 tick을 반드시 표시해서 사용자에게 오차를 보여줘야 한다.
- `'hidden'` 넘치면 표시하지 않는다.
- `'pull'` 차트나 분할 경계를 넘어선 만큼 끌어당겨서 표시한다.<br/>  실제 표시 위치와 달라지므로 tick을 반드시 표시해서 사용자에게 오차를 보여줘야 한다.

<DefaultValue value="'pull'" contents=""></DefaultValue>

### rows: `number{:js}`[#rows]
수평 축일 때 tick label 배치 행 수.<br/>
1은 한 줄, 2면 두 줄 등으로 여러 줄로 나눠서 표시한다.  
<DefaultValue value="0" contents=""></DefaultValue>
### startStep: `number{:js}`[#startStep]
step이 2 이상이 될 때, 표시가 시작되는 label 위치.<br/>
0 ~ [step](#step) - 1 사이의 값으로 지정한다.
<Content type="expression" value="'category'"></Content> 축에서 의미있게 사용할 수 있다.  
<DefaultValue value="0" contents=""></DefaultValue>
### step: `number{:js}`[#step]
label 표시 간격.<br/>
1이면 모든 tick 표시. 2이면 하나씩 건너 띄어서 표시.
2 이상일 때 [startStep](#startstep) 으로 지정된 step부터 배치된다.  
<DefaultValue value="0" contents=""></DefaultValue>

### textCallback[#textCallback]
축 tick 라벨에 표시될 텍스트를 리턴한다.<br/>
<Content type="keyword" value="undefined"></Content>나 <Content type="keyword" value="null"></Content>을 리턴하면 [text](#text) 속성 등에 설정된 값으로 표시하거나,
값에 따라 자동 생성되는 텍스트를 사용한다.
빈 문자열 등 정상적인 문자열을 리턴하면 그 문자열대로 표시된다. [prefix](#prefix) 나 포맷 속성 등은 적용되지 않는다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### useSymbols: `boolean | '*'{:js}`[#useSymbols]
<Content type="keyword" value="true"></Content>로 지정하면 label 끝에 단위 문자를 추가한다.<br/>
표시되는 값들의 연속성을 위해 <Content type="keyword" value="true"></Content>로 지정해도 step 간격이 1000 이상일 때 추가하는데,
간격과 상관없이 단위 문자를 표시하려면 <Content type="expression" value="'*'"></Content>으로 지정한다.  
<DefaultValue value="true" contents=""></DefaultValue>
