## flag
축 상에 crosshair의 정보를 표시하는 view.
## Properties

### format: `string{:javascript}`[#format]
flag에 표시될 텍스트 형식.


별도로 지정하지 않으면 현재 위치에 해당하는 축 값을 표시한다.
Category 축인 경위 위치에 해당하는 category 이름을 표시한다.  

### style: `SVGStyles | string{:javascript}`[#style]
스타일셋 혹은 class selector.  

### textStyles: `SVGStyles | string{:javascript}`[#textStyles]
flag에 표시되는 text의 스타일.  

### visible: `boolean{:javascript}`[#visible]
표시 여부.\
모델에 따라 기본값이 다르다.
