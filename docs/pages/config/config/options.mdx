## **options**

## Properties

### animatable: `boolean{:js}`[#animatable]
<Content type="keyword" value="false"></Content>로 지정하면 차트 전체척으로 animation 효과를 실행하지 않는다.  
<DefaultValue value="true" contents=""></DefaultValue>
### axisGap: `number{:js}`[#axisGap]
복수 axis가 표시되는 경우 axis 사이의 간격  
<DefaultValue value="8" contents="pixels"></DefaultValue>
### [credits](./options/credits)
크레딧 모델.  

### [pointHovering](./options/pointHovering)
데이터포인트에 마우수가 올라갈 때 처리하는 방식.<br/>  

### seriesHovering: `boolean{:js}`[#seriesHovering]
데이터포인트에 마우스가 올라가면 나머지 시리즈들을 반투명 처리해서 연결된 데이터포인터의 시리즈를 강조한다.  
<DefaultValue value="false" contents=""></DefaultValue>
### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fillOpacity | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| rx | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeDasharray | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |  
| textAlign | `'center' \| 'left' \| 'right'{:js}` | true |<DefaultValue value="undefined" contents=""></DefaultValue>
### visible: `boolean{:js}`[#visible]
표시 여부.  
<DefaultValue value="true" contents=""></DefaultValue>
### xStart: `any{:js}`[#xStart]
x축 값이 설정되지 않은 시리즈 첫번째 데이터 point에 설정되는 x값.<br/>
이 후에는 [xStep](#xstep) 씩 증가시키면서 설정한다.
시리즈의 [xStart](/config/config/options#xstart) 가 설정되면 그 값이 사용된다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### xStep: `string | number{:js}`[#xStep]
x축 값이 설정되지 않은 데이터 point에 지정되는 x값의 간격.

첫번째 값은 [xStart](#xstart) 로 설정한다.
time 축일 때, 정수 값 대신 시간 단위(<Content type="expression" value="'y', 'm', 'd', 'h', 'n', 's'"></Content>)로 지정할 수 있다.
시리즈의 [xStep](/config/config/options#xstep) 이 설정되면 그 값이 사용된다.  
<DefaultValue value="1" contents=""></DefaultValue>
