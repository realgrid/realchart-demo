## Properties

### align[#align]


`default: Align.RIGHT`  

### floating: `boolean{:javascript}`[#floating]
true이면  verticalAlign 이 'top', 'bottom'일 때도,
별도의 영역을 차지하지 않고 chart view위에 표시된다.  
`default: false`  

### offsetX: `number{:javascript}`[#offsetX]
align 으로 지정된 수평 위치에서, 양수로 지정하면 안쪽으로 음수면 바깥쪽으로 밀어서 표시한다.  
`default: 2`  

### offsetY: `number{:javascript}`[#offsetY]
verticalAlign 으로 지정된 수직 위치에서, 양수로 지정하면 안쪽으로 음수면 바깥쪽으로 밀어서 표시한다.  
`default: 1`  

### style: `SVGStyles | string{:javascript}`[#style]
스타일셋 혹은 class selector.  

### text: `string{:javascript}`[#text]
표시할 문자열.  
`default: 'RealChart` v1.0' 

### url: `string{:javascript}`[#url]
이 속성을 지정하면 click시 해당 url로 이동한다.  
`default: 'http://realgrid.com'`  

### verticalAlign[#verticalAlign]


`default: VerticalAlign.BOTTOM`  

### visible: `boolean{:javascript}`[#visible]
표시 여부.\
모델에 따라 기본값이 다르다.
- [Legend Visible](https:/jsfiddle.net/gh/get/library/pure/realgrid/realchart-demo/tree/master/common/legend-visible)
