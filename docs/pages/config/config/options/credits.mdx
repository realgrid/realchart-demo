## Properties

### align: `'center'|'left'|'right'{:js}`[#align]
- `'center'` 
- `'left'` 
- `'right'` 

<DefaultValue value="'right'" contents=""></DefaultValue>
### floating: `boolean{:js}`[#floating]
<Content type="keyword" value="true"></Content>이면  [verticalAlign](#verticalalign) 이 <Content type="expression" value="'top', 'bottom'"></Content>일 때도,
별도의 영역을 차지하지 않고 chart view위에 표시된다.  
<DefaultValue value="false" contents=""></DefaultValue>
### offsetX: `number{:js}`[#offsetX]
[align](#align) 으로 지정된 수평 위치에서, 양수로 지정하면 안쪽으로 음수면 바깥쪽으로 밀어서 표시한다.  
<DefaultValue value="2" contents=""></DefaultValue>
### offsetY: `number{:js}`[#offsetY]
[verticalAlign](#verticalalign) 으로 지정된 수직 위치에서, 양수로 지정하면 안쪽으로 음수면 바깥쪽으로 밀어서 표시한다.  
<DefaultValue value="1" contents=""></DefaultValue>
### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fillOpacity | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| rx | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeDasharray | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |  
| textAlign | `'center' \| 'left' \| 'right'{:js}` | true |
