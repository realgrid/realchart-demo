## legend
차트 범례 설정 모델.
visible 기본값이 undefined이고,
따로 지정하지 않으면 시리즈가 둘 이상 포함돼야 legend를 표시한다.
## Properties

### alignBase: `'chart'|'parent'|'plot'{:js}`[#alignBase]
legend 정렬 기준.  
- `'chart'` 
- `'parent'` 상위 모델이 존재하는 경우 상위 모델 영역 기준.\  상위가 없으면 기본값(대부분 'plot')과 동일.  ex) subtitle인 경우 title 기준.
- `'plot'` 

default: <span style={{ color: 'var(--shiki-token-string-expression)', fontWeight: 700 }}>'plot'</span>
### backgroundStyle: `SVGStyles | string{:js}`[#backgroundStyle]
배경 스타일 셋.
배경 색상 및 경계선 스타일을 지정할 수 있다.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fillOpacity | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeDasharray | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |
### gap: `number{:js}`[#gap]
**[location](#location)** 이 'plot'이 아닐 때,
legend view와 나머지 chart 영역 사이의 gap.  
default: <span style={{ color: 'var(--shiki-color-text)', fontWeight: 700 }}>6</span>
### itemGap: `number{:js}`[#itemGap]
legend 아이템들 사이의 간격.  
default: <span style={{ color: 'var(--shiki-color-text)', fontWeight: 700 }}>8</span>
### itemsAlign: `'center'|'end'|'start'{:js}`[#itemsAlign]
한 라인의 item들이 배치되는 위치.  
- `'center'` 수평 혹은 수직의 중앙으로 몰아서 배치한다.
- `'end'` 수평일 때 오른쪽, 수직일 때는 아래쪽으로 몰아서 배치한다.
- `'start'` 수평일 때 왼쪽, 수직일 때는 위쪽으로 몰아서 배치한다.

default: <span style={{ color: 'var(--shiki-token-string-expression)', fontWeight: 700 }}>'center'</span>
### itemsPerLine: `number{:js}`[#itemsPerLine]
한 줄 당 표시할 최대 legend 항목 수.  

### layout: `'auto'|'horizontal'|'vertical'{:js}`[#layout]
item 배치 방향.  
- `'auto'` legend가 차트 좌우에 배치되면 item들을 수직으로 배치.  legend가 차트 상하에 배치되면 item들을 수평으로 배치.
- `'horizontal'` item들을 수평으로 배치
- `'vertical'` item들을 수직으로 배치

default: <span style={{ color: 'var(--shiki-token-string-expression)', fontWeight: 700 }}>'auto'</span>
### lineGap: `number{:js}`[#lineGap]
라인 사이의 간격.  
default: <span style={{ color: 'var(--shiki-color-text)', fontWeight: 700 }}>4</span>
### location: `'bottom'|'left'|'plot'|'right'|'top'{:js}`[#location]
표시 위치.  
- `'bottom'` 차트 본체 아래 표시한다.
- `'left'` 차트 본체 왼쪽에 표시한다.
- `'plot'` 차트 본체 영역 내부에 표시한다.
- `'right'` 차트 본체 오른쪽에 표시한다.
- `'top'` 차트 타이틀 아래 표시한다.

default: <span style={{ color: 'var(--shiki-token-string-expression)', fontWeight: 700 }}>'bottom'</span>
### markerGap: `number{:js}`[#markerGap]
marker와 text사이의 간격.  
default: <span style={{ color: 'var(--shiki-color-text)', fontWeight: 700 }}>4</span>
### markerSize: `number{:js}`[#markerSize]
marker 크기.  
default: <span style={{ color: 'var(--shiki-color-text)', fontWeight: 700 }}>10</span> 픽셀.
### maxHeight: `string | number{:js}`[#maxHeight]
수직  **[배치](#layout)** 일 때,
최대 높이를 픽셀 단위의 크기 혹은 plot 높이에 대한 상대 길이를 '%'로 지정한다.  

### maxWidth: `string | number{:js}`[#maxWidth]
수평  **[배치](#layout)** 일 때,
최대 너비를 픽셀 단위의 크기 혹은 plot 너비에 대한 상대 길이를 '%'로 지정한다.  

### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fillOpacity | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeDasharray | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |
### visible: `boolean{:js}`[#visible]
표시 여부.  
명시적으로 true로 설정되거나, 명시적 false가 아니면서 표시 항목 수가 1보다 클 때 표시된다..\

default: <span style={{ color: 'var(--shiki-token-keyword)', fontWeight: 700 }}>undefined</span>
