## legend
차트 범례 설정 모델.
visible 기본값이 undefined이고,
따로 지정하지 않으면 시리즈가 둘 이상 포함돼야 legend를 표시한다.
## Properties

### alignBase
legend 정렬 기준.  
`default: AlignBase.PLOT`  
- `'chart'` 
- `'plot'` 
### backgroundStyles
배경 스타일 셋.
배경 색상 및 경계선 스타일을 지정할 수 있다.  

### bottom: `number`
position 이  plot 일 때, plot 영역의 하단 모서리와 legend의 간격.
 top 이 지정되면 이 속성은 무시된다.  

### gap: `number`
legend view와 나머지 chart 영역 사이의 gap.  
`default: 6`  

### itemGap: `number`
legend 아이템들 사이의 간격.  
`default: 8`  

### itemsAlign
한 라인의 item들이 배치되는 위치.  
`default: LegendItemsAlign.CENTER`  
- `'center'` 수평 혹은 수직의 중앙으로 몰아서 배치한다.
- `'end'` 수평일 때 오른쪽, 수직일 때는 아래쪽으로 몰아서 배치한다.
- `'start'` 수평일 때 왼쪽, 수직일 때는 위쪽으로 몰아서 배치한다.
### itemsPerLine: `number`
한 줄 당 표시할 최대 legend 항목 수.  

### layout
item 배치 방향.  
`default: LegendLayout.AUTO`  
- `'auto'` legend가 차트 좌우에 배치되면 item들을 수직으로 배치.  legend가 차트 상하에 배치되면 item들을 수평으로 배치.
- `'horizontal'` item들을 수평으로 배치
- `'vertical'` item들을 수직으로 배치
### left: `number`
position 이  plot 일 때, plot 영역의 좌측 모서리와 legend의 간격.  
`default: 10`  

### lineGap: `number`
라인 사이의 간격.  
`default: 4`  

### markerGap: `number`
marker와 text사이의 간격.  
`default: 4`  

### maxHeight
수직  배치 일 때,
최대 높이를 픽셀 단위의 크기 혹은 plot 높이에 대한 상대 길이를 '%'로 지정한다.  

### maxWidth
수평  배치 일 때,
최대 너비를 픽셀 단위의 크기 혹은 plot 너비에 대한 상대 길이를 '%'로 지정한다.  

### position
legned 표시 위치.  
`default: LegendPosition.BOTTOM`  
- `'bottom'` 차트 본체 아래 표시한다.
- `'left'` 차트 본체 왼쪽에 표시한다.
- `'plot'` 차트 본체 영역 내부에 표시한다.
- `'right'` 차트 본체 오른쪽에 표시한다.
- `'top'` 차트 타이틀 아래 표시한다.
### right: `number`
position 이  plot 일 때, plot 영역의 우측 모서리와 legend의 간격.
 left 가 지정되면 이 속성은 무시된다.  

### top: `number`
position 이  plot 일 때, plot 영역의 상단 모서리와 legend의 간격.  
`default: 10`
