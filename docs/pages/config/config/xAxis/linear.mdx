선형 연속 축.
값 사아의 비율과 축 길이 비율이 항상 동일한 축.

### [label](./linear/label)
[겹치는 경우가 발생할 때]
1. step이 0보다 큰 값으로 설정되면 반영한다.
2. staggerRows가 0보다 큰 값으로 설정되면 반영한다.
3. rotation이 0이 아닌 명시적 값으로 설정되면 반영한다.
4. 1~3 모두 설정되지 않은 경우 autoArrange 설정에 따라 자동 배치한다.
5. 배치 후 공간을 초과하는 label은 wrap 속성에 따라 줄나누기를 하거나, 
   ellipsis('...')로 처리해서 표시한다.

### maxPadding: `number{:js}`[#maxPadding]
마지막 tick 뒤쪽에 추가되는 최소 여백을 축 길이에 대한 상대값으로 지정한다.
이 값을 지정하지 않으면  **[padding](#padding)** 에 지정된 값을 따른다.
 **[endFit](#endfit)** 이  **[TICK](/docs/api/classes/TICK)** 일 때,
data point의 최대값과 마지막 tick 사이에 이미 그 이상의 간격이 존재한다면 무시된다.
 **[strictMax](#strictmax)** 가 지정되거나,  **[maxValue](#maxvalue)** 가 계산된 최대값보다 큰 경우에도 이 속성은 무시된다.  

### maxValue: `number{:js}`[#maxValue]
명시적으로 지정하는 최대값.  
명시적으로 지정하는 최대값.\
축에 연결된 data point들의 값으로 계산된 최대값보다 이 속성 값이 크면 대신 이 값이 축의 최대값이 되고,
 **[maxPadding](#maxpadding)** 도 무시된다.
계산값이 더 크면 이 속성은 무시된다.\
계산값과 무관하게 최대값을 지정하려면  **[strictMax](#strictmax)** 을 사용한다.  

### minPadding: `number{:js}`[#minPadding]
첫번째 tick 앞쪽에 추가되는 최소 여백을 축 길이에 대한 상대값으로 지정한다.
이 값을 지정하지 않으면  **[padding](#padding)** 에 지정된 값을 따른다.
 **[startFit](#startfit)** 이  **[TICK](/docs/api/classes/TICK)** 일 때,
data point의 최소값과 첫번째 tick 사이에 이미 그 이상의 간격이 존재한다면 무시된다.
 **[strictMin](#strictmin)** 가 지정되거나,  **[minValue](#minvalue)** 가 계산된 최소값보다 작은 경우에도 이 속성은 무시된다.  

### minValue: `number{:js}`[#minValue]
명시적으로 지정하는 최소값.

### position: `'base'|'normal'|'opposite'{:js}`[#position]
표시 위치.
기본적으로 상대 축의 원점 쪽에 표시된다.  
- `'base'` 상대 축의 baseValue 지점에 표시된다.      [주의]   1. 축에 연결된 시리즈들이 BarSeries 계열일 때만 가능하다.  2. 차트의 X축 하나에만 적용할 수 있다. 두번째로 지정된 축의 속성은  `normal` 로 적용된다.  3. 상대 축이 **linear** 가 아니거나  [baseValue](/config/config/xAxis#basevalue) 가 min 보다 작거나 max보다 크면 이 값은 무시되고,      `normal` 로 적용된다.
- `'normal'` X축은 아래쪽에 수평으로, Y축은 왼쪽에 수직으로 표시된다.       [inverted](/config/config#inverted) 이면 Y축이 아래쪽에 수평으로, X축은 왼쪽에 수직으로 표시된다.
- `'opposite'` X축은 위쪽에 수평으로, Y축은 오른쪽에 수직으로 표시된다.       [inverted](/config/config#inverted) 이면 Y축이 위쪽에 수평으로, X축은 오른쪽에 수직으로 표시된다.

`default: 'normal'`  

### reversed: `boolean{:js}`[#reversed]
true면 반대 방향으로 point 위치들이 지정된다.  
`default: false`  

### [scrollBar](./linear/scrollBar)
축 스크롤바 모델.\
축 스크롤 상태를 표시하고, 사용자가 스크롤 범위나 위치를 변경할 수 있다.  

### startFit[#startFit]
축 시작 위치에 tick 표시 여부.
 **[strictMin](#strictmin)** 이 설정되면  `value` 로 적용된다.  


`default: AxisFit.DEFAULT`  

### strictMax: `number{:js}`[#strictMax]
무조건 적용되는 최대값.
즉, 이 값보다 큰 값을 갖는 시리즈 포인트들은 표시되지 않는다.
maxPadding도 적용되지 않는다.  

### strictMin: `number{:js}`[#strictMin]
무조건 적용되는 최소값.
즉, 이 값보다 작은 값을 갖는 시리즈 포인트들은 표시되지 않는다.
minPadding도 적용되지 않는다.
