## **tick**
축에 표시되는 tick 위치와 표시 마크에 관한 설정 모델.
## Properties

### gap: `number{:js}`[#gap]
tick 라인과 다른 요소(label이나 title) 사이의 간격.  
<DefaultValue value="3" contents=""></DefaultValue>
### length: `number{:js}`[#length]
axis tick line length.  
<DefaultValue value="7" contents=""></DefaultValue>
### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### visible: `boolean{:js}`[#visible]
표시 여부.  
<DefaultValue value="true" contents=""></DefaultValue>
