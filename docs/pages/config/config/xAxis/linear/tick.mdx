## [tick](/config/config/tick)

## Properties

### gap: `number{:js}`[#gap]
tick 라인과 다른 요소(label이나 title) 사이의 간격.  
default: <span style={{ color: 'var(--shiki-color-text)', fontWeight: 700 }}>3</span>
### length: `number{:js}`[#length]
axis tick line length.  
default: <span style={{ color: 'var(--shiki-color-text)', fontWeight: 700 }}>7</span>
### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fillOpacity | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeDasharray | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |
### visible: `boolean{:js}`[#visible]
표시 여부.  
default: <span style={{ color: 'var(--shiki-token-keyword)', fontWeight: 700 }}>true</span>
