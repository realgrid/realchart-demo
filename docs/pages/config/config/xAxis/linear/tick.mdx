## **tick**
축에 표시되는 tick 위치와 표시 마크에 관한 설정 모델.
## Properties

### baseAxis: `string | number{:js}`[#baseAxis]
tick 개수를 맞춰야 하는 대상 axis.<br/>
base의 strictMin, strictMax가 설정되지 않아야 한다.
base의 startFit, endFilt의 `tick` 으로 설정되어야 한다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### baseRange: `boolean{:js}`[#baseRange]
[baseAxis](#baseaxis) 가 지정된 경우, <Content type="keyword" value="true"></Content>로 지정하면 base 축의 범위와 동일하게
축의 최소/최대가 결정된다.  
<DefaultValue value="false" contents=""></DefaultValue>
### gap: `number{:js}`[#gap]
tick 라인과 다른 요소(label이나 title) 사이의 간격.  
<DefaultValue value="3" contents=""></DefaultValue>
### length: `number{:js}`[#length]
axis tick line length.  
<DefaultValue value="7" contents=""></DefaultValue>
### stepInterval: `string | number{:js}`[#stepInterval]
tick들 사이의 값 간격.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### stepPixels: `number{:js}`[#stepPixels]
tick들 사이의 대략적인 픽셀 간격.<br/>
다른 설정이 없다면 이 값을 기준으로 tick 개수가 결정된다.  
<DefaultValue value="72" contents=""></DefaultValue>
### steps: `number[]{:js}`[#steps]
명시적으로 설정하는 setp 목록<br/>
양 끝을 <Content type="keyword" value="NaN"></Content>으로 지정하면 계산된 min/max로 설정된다.
이 목록이 설정되면 스텝 설정과 관련된 다른 속성들은 무시된다.<b/> [stepCallback](#stepcallback) 이 지정되면 이 속성은 무시된다.  
<DefaultValue value="undefined" contents=""></DefaultValue>

| strokeWidth | `string{:js}` | true |  
| textAlign | `'center' \| 'left' \| 'right'{:js}` | true |<DefaultValue value="undefined" contents=""></DefaultValue>
### visible: `boolean{:js}`[#visible]
표시 여부.  
<DefaultValue value="true" contents=""></DefaultValue>
