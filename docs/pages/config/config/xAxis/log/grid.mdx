## **grid**
Axis tick의 위치에 수평 혹은 수직선으로 plot 영역을 구분 표시한다.
 [visible](#visible) 기본값이 <Content type="keyword" value="undefined"></Content>인데,
visible이 <Content type="keyword" value="undefined"></Content>나 <Content type="keyword" value="null"></Content>로 지정되면, 축 위치에 따라 visible 여부가 결정된다.
## Properties

### firstVisible: `boolean{:js}`[#firstVisible]
시작 값에 표시되는 그리드 선을 표시할 지 여부.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### lastVisible: `boolean{:js}`[#lastVisible]
끝 값에 표시되는 그리드 선을 표시할 지 여부.  
<DefaultValue value="y축이면" contents="true."></DefaultValue>
### [rows](./grid/rows)
축 그리드 사이에 생성된 영역 표시 설정 모델.  

### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  
- `SVGStyles{:js}` 

| Name | Type | Optional |

| fill | `string{:js}` | true |  
| fillOpacity | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| rx | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeDasharray | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |  
| textAlign | `'center' \| 'left' \| 'right'{:js}` | true |<DefaultValue value="undefined" contents=""></DefaultValue>
### visible: `boolean{:js}`[#visible]
표시 여부.
