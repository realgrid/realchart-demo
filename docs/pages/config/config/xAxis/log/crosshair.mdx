## crosshair

## Properties

### [flag](./log/flag)
축 상에 crosshair의 정보를 표시하는 view 설정 모델.  

### followPointer: `boolean{:javascript}`[#followPointer]
false면 type이 'line'인 경우 데이터포인트나 카테고리 단위로 이동한다.  
`default: true`  

### showAlways: `boolean{:javascript}`[#showAlways]
true면 마우스 위치를 따라 항상 표시하고, false면 data point 위에 마우스가 있을 때만 표시한다.  
`default: true`  

### style: `SVGStyles | string{:javascript}`[#style]
스타일셋 혹은 class selector.  

### type[#type]
표시 방식.  
- `'auto'` 카테고리 축이면 bar, 연속 축이면 line으로 표시한다.
- `'line'` 항상 line으로 표시한다.

`default: CrosshairType.AUTO`  

### visible: `boolean{:javascript}`[#visible]
표시 여부.\
모델에 따라 기본값이 다르다.
