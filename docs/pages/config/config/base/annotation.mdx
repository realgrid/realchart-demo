## **annotation**
Annotation 모델.<br/>
## Properties

### align: `'center'|'left'|'right'{:js}`[#align]
수평 배치.<br/>  
- `'center'` 
- `'left'` 
- `'right'` 

<DefaultValue value="'left'" contents="anchor가 지정되면 'center', 아니면 'left'"></DefaultValue>
### anchor: `string{:js}`[#anchor]
어노테이션 배치 기준이 되는 차트 구성 요소.<br/>
현재, 같은 영역(body 혹은 chart)에 포함된 [게이지](/config/config/base/gauge) 나 다른 어노테이션의 [이름](#name) 을 지정할 수 있다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### backgroundStyle: `SVGStyles | string{:js}`[#backgroundStyle]
배경 스타일.<br/>
경계 및 배경 색, padding 스타일을 지정할 수 있다.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fillOpacity | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| rx | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeDasharray | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |  
| textAlign | `'center' \| 'left' \| 'right'{:js}` | true |<DefaultValue value="undefined" contents=""></DefaultValue>
### front: `boolean{:js}`[#front]
<Content type="keyword" value="true"></Content>로 지정하면 시리즈들 위에 표시된다.  
<DefaultValue value="false" contents=""></DefaultValue>
### height: `string | number{:js}`[#height]
Annotation 높이.<br/>
픽셀 단위의 고정 값이나, plot 영역에 대한 상태 크기롤 지정할 수 있다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### name: `string{:js}`[#name]
어노테이션 이름.<br/>
동적으로 어노테이션을 다루기 위해서는 반드시 지정해야 한다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### noClip: `boolean{:js}`[#noClip]
<Content type="keyword" value="true"></Content>로 지정하면 상위 영역을 벗어난 부분도 표시되게 한다.<br/>
body에 포함된 경우 body 영역,
chart에 포함되고 scope이 <Content type="expression" value="'chart'"></Content>일 때 chart 영역.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### offsetX: `string | number{:js}`[#offsetX]
[align](#align) 과 [verticalAlign](#verticalalign) 으로 지정된 위치에서 실제 표시될 위치의 수평 간격.<br/>
값이 양수일 때, [anchor](#anchor) 가 지정된 경우 anchor 아이템으 밖으로 멀어지고, 아니면 영역 경계 안쪽으로 멀어진다.
또, [anchor](#anchor) 가 지정된 경우 **<Content type="expression" value="'0.5w'"></Content>** 등으로 이 어노테이션의 너비를 기준으로 한 크기로 지정할 수 있다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### offsetY: `string | number{:js}`[#offsetY]
[align](#align) 과 [verticalAlign](#verticalalign) 으로 지정된 위치에서 실제 표시될 위치의 수직 간격.<br/>
값이 양수일 때, [anchor](#anchor) 가 지정된 경우 anchor 아이템으 밖으로 멀어지고, 아니면 영역 경계 안쪽으로 멀어진다.
또, [anchor](#anchor) 가 지정된 경우 **<Content type="expression" value="'0.5h'"></Content>**처럼 이 어노테이션의 너비를 기준으로 한 크기로 지정할 수 있다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### rotation: `number{:js}`[#rotation]
회전 각도.<br/>
0 ~ 360 사이의 값으로 지정한다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### scope: `'chart'|'container'{:js}`[#scope]
어노테이션 배치 기준.<br/>
[주의]body에 설정된 annoation에는 적용되지 않는다.  
- `'chart'` container에서 padding을 적용한 영역을 기준으로 표시한다.
- `'container'` container 전체 영역을 기준으로 표시한다.

<DefaultValue value="'chart'" contents=""></DefaultValue>
