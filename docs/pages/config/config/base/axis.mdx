## **axis**
축 모델.

차트에서 축을 명식적으로 지정하지 않으면, 첫번째 시리즈에 합당한 축이 기본 생성된다.
 [축 개요](/guide/axes) 페이지를 참조한다.
## Properties

### col: `number{:js}`[#col]
분할 모드일 때 축이 표시될 pane의 수평 위치.  
<DefaultValue value="0" contents=""></DefaultValue>
### [crosshair](./axis/crosshair)
직선 또는 bar 형태로 축 위의 마우스 위치를 표시하는 구성 요소 모델.  

### [grid](./axis/grid)
visible 기본값이 <Content type="keyword" value="undefined"></Content>이다.
visible이 <Content type="keyword" value="undefined"></Content>나 <Content type="keyword" value="null"></Content>로 지정되면, 축 위치에 따라 visible 여부가 결정된다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### [guides](./axis/guides)
기본적으로 이 축에 연결된 모든 body에 모두 표시된다.
col, row를 지정해서 특정 body에만 표시되도록 할 수 있다.

### position: `'base'|'normal'|'opposite'{:js}`[#position]
표시 위치.
기본적으로 상대 축의 원점 쪽에 표시된다.  
- `'base'` 상대 축의 baseValue 지점에 표시된다.      [주의]   1. 축에 연결된 시리즈들이 BarSeries 계열일 때만 가능하다.  2. 차트의 X축 하나에만 적용할 수 있다. 두번째로 지정된 축의 속성은 [NORMAL](#normal) 로 적용된다.  3. 상대 축이 **linear** 가 아니거나 [baseValue](/config/config/xAxis#basevalue) 가 min 보다 작거나 max보다 크면 이 값은 무시되고, [NORMAL](#normal) 로 적용된다.
- `'normal'` X축은 아래쪽에 수평으로, Y축은 왼쪽에 수직으로 표시된다.     [inverted](/config/config#inverted) 이면 Y축이 아래쪽에 수평으로, X축은 왼쪽에 수직으로 표시된다.
- `'opposite'` X축은 위쪽에 수평으로, Y축은 오른쪽에 수직으로 표시된다.     [inverted](/config/config#inverted) 이면 Y축이 위쪽에 수평으로, X축은 오른쪽에 수직으로 표시된다.

<DefaultValue value="'normal'" contents=""></DefaultValue>

### [sectorLine](./axis/sectorLine)

<DefaultValue value="undefined" contents=""></DefaultValue>

### tooltipFooter: `string{:js}`[#tooltipFooter]

<DefaultValue value="undefined" contents=""></DefaultValue>
### tooltipHeader: `string{:js}`[#tooltipHeader]

tooltipRow,
...

### tooltipRow: `string{:js}`[#tooltipRow]

### totalAngle: `number{:js}`[#totalAngle]
Polar 차트에서 사용될 때 원호 전체 각도.
0 ~ 360 사이의 값으로 지정해야 한다.
범위를 벗어난 값은 범위 안으로 조정된다.
지정하지 않거나 잘못된 값이면 360으로 계산된다.  
<DefaultValue value="360" contents=""></DefaultValue>

<DefaultValue value="undefined" contents=""></DefaultValue>

표시 여부.  
<DefaultValue value="true" contents=""></DefaultValue>
