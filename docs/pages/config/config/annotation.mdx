## annotation
차트 보조 요소 설정
## Properties

### align[#align]
수평 배치.  


`default: Align.LEFT`  

### front: `boolean{:js}`[#front]
true로 지정하면 시리즈들 위에 표시된다.  
`default: false`  

### name: `string{:js}`[#name]
Annotation 이름.
동적으로 Annotation을 다루기 위해서는 반드시 지정해야 한다.  

### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fillOpacity | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |
### visible: `boolean{:js}`[#visible]
표시 여부.  
`default: true`
