## **export**
차트 내보내기 설정 모델
## Properties

### fileName: `string{:js}`[#fileName]
내보내기 버튼을 사용해 내보낸 차트에 사용할 확장자 없는 파일 이름입니다.
export 함수를 호출할 때 fileName을 지정하지 않을 경우 이 속성이 적용되니다.
fileName을 지정하지 않을 경우 realchart.type으로 다운로드 됩니다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### hideNavigator: `boolean{:js}`[#hideNavigator]
<Content type="keyword" value="true"></Content>로 지정하면 내보내기 결과에  가 포함되지 않는다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
