## title
차트 타이틀 설정 모델.
## Properties

### alignBase[#alignBase]
정렬 기준.  
- `'chart'` 
- `'plot'` 

`default: AlignBase.PLOT`  

### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  

### text: `string{:js}`[#text]
제목 텍스트.  
`default: 'Title'`  

### visible: `boolean{:js}`[#visible]
표시 여부.\  
`default: true`
