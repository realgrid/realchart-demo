## widget
Widget 모델.
기본적으로 plot 영역에 표시된다.
plot 영역에 표시될 때  left ,  top  등으로 위치를 지정할 수 있다.
## Properties

### bottom
position 이  LegendPosition.PLOT plot 일 때, plot 영역의 하단 모서리와 legend의 간격.
 top 이 지정되면 이 속성은 무시된다.  

### left
position 이  LegendPosition.PLOT plot 일 때, plot 영역의 좌측 모서리와 legend의 간격.  
`default: 10`  

### right
position 이  LegendPosition.PLOT plot 일 때, plot 영역의 우측 모서리와 legend의 간격.
 left 가 지정되면 이 속성은 무시된다.  

### style
스타일셋 혹은 class selector.  

### top
position 이  LegendPosition.PLOT plot 일 때, plot 영역의 상단 모서리와 legend의 간격.  
`default: 10`
