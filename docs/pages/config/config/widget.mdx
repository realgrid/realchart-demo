## widget
Widget 모델.
기본적으로 plot 영역에 표시된다.
plot 영역에 표시될 때  left ,  top  등으로 위치를 지정할 수 있다.
## Properties

### style: `SVGStyles | string{:javascript}`[#style]
스타일셋 혹은 class selector.  

### visible: `boolean{:javascript}`[#visible]
표시 여부.\
모델에 따라 기본값이 다르다.
