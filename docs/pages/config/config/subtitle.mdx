## subtitle
차트 sub 타이틀 설정 모델.
## Properties

### alignBase
정렬 기준.  
- `'chart'` 
- `'plot'` 

`default: AlignBase.PLOT`  

### style: `SVGStyles | string{:typescript}`
스타일셋 혹은 class selector.  

### text: `string{:typescript}`
부제목 텍스트  
`default: ''`  

### visible: `boolean{:typescript}`
표시 여부.
모델에 따라 기본값이 다르다.
