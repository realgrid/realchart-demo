## subtitle
차트 sub 타이틀 설정 모델.
## Properties

### alignBase[#alignBase]
정렬 기준.  
- `'chart'` 
- `'plot'` 

`default: AlignBase.PLOT`  

### style: `SVGStyles | string{:javascript}`[#style]
스타일셋 혹은 class selector.  

### text: `string{:javascript}`[#text]
부제목 텍스트  
`default: ''`  

### visible: `boolean{:javascript}`[#visible]
표시 여부.\
모델에 따라 기본값이 다르다.
- [Legend Visible](https:/jsfiddle.net/gh/get/library/pure/realgrid/realchart-demo/tree/master/common/legend-visible)
