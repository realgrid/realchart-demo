## subtitle
차트 sub 타이틀 설정 모델.
## Properties

### alignBase: `'chart'|'plot'{:js}`[#alignBase]
정렬 기준.  
- `'chart'` 
- `'plot'` 

`default: AlignBase.PLOT`  

### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  

### text: `string{:js}`[#text]
부제목 텍스트  
`default: ''`  

### visible: `boolean{:js}`[#visible]
표시 여부.\  
`default: true`
