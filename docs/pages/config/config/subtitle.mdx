## **subtitle**
차트 sub 타이틀 설정 모델.
## Properties

### align: `'center'|'left'|'right'{:js}`[#align]
정렬.  
<DefaultValue value="'center'" contents=""></DefaultValue>
### alignBase: `'chart'|'parent'|'plot'{:js}`[#alignBase]
정렬 기준.  
<DefaultValue value="'plot'" contents=""></DefaultValue>
### backgroundStyle: `SVGStyles | string{:js}`[#backgroundStyle]
배경 스타일 셋.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### gap: `number{:js}`[#gap]
주 제목과 부 제목이 표시되는 영역과 차트 본체 등 과의 간격.

주 제목이 표시되면 (부 제목의 값은 무시되고)주 제목의 값을 사용하고,
부 제목만 표시될 때는 부 제목의 값을 사용한다.  
<DefaultValue value="10" contents=""></DefaultValue>
### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### text: `string{:js}`[#text]
부제목 텍스트  
<DefaultValue value="''" contents=""></DefaultValue>
### titleGap: `number{:js}`[#titleGap]
주 제목과 사이의 간격.  
<DefaultValue value="2" contents=""></DefaultValue>
### visible: `boolean{:js}`[#visible]
표시 여부.  
<DefaultValue value="true" contents=""></DefaultValue>
