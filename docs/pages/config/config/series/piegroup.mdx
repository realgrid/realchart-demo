## series[type=piegroup]

## Properties

### innerSize: `string | number{:javascript}`[#innerSize]
Pie 시리즈들의 그룹이고, 
 layout 이  SeriesGroupLayout.FILL 이나  SeriesGroupLayout.STACK 인 경우,
경우 0보다 큰 값을 지정해서 도넛 형태로 표시할 수 있다.


포함된 pie 시리즈들의 innerSize는 무시된다.  
`default: 0`  

### layout[#layout]


`default: SeriesGroupLayout.DEFAULT`  

### layoutMax: `number{:javascript}`[#layoutMax]
layout 이  SeriesGroupLayout.FILL 일 때 상대적 최대값.

  
`default: 100`  

### polarSize: `string | number{:javascript}`[#polarSize]
polar 그룹일 때 원형 플롯 영역의 크기.


픽셀 크기나 차지할 수 있는 전체 크기에 대한 상대적 크기로 지정할 수 있다.


Pie 시리즈들의 그룹이고, 
 layout 이  SeriesGroupLayout.FILL 이나  SeriesGroupLayout.STACK 인 경우
개별 시리즈의 size 대신 이 속성값으로 표시되고,
각 시리즈의 size는 상대 크기로 적용된다.  
`default: '80%'`  

### style: `SVGStyles | string{:javascript}`[#style]
스타일셋 혹은 class selector.  

### visibleInLegend: `boolean{:javascript}`[#visibleInLegend]
명시적 false로 지정하면 legend에 표시하지 않는다.  
`default: true`  

### xAxis: `string | number{:javascript}`[#xAxis]

### yAxis: `string | number{:javascript}`[#yAxis]

### visible: `boolean{:javascript}`[#visible]
표시 여부.\
