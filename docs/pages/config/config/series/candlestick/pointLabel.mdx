## **pointLabel**
Series data point label options.
## Properties

### autoContrast: `boolean{:js}`[#autoContrast]
텍스트가 data point 내부에 표시되는 경우 포인트 색상과 대조되도록 표시한다.


밝게 표시할 때는 [lightStyle](#lightstyle) 을 적용하고,
어둡게 표시할 때는 [darkStyle](#darkstyle) 이 적용된다.
brightStyle이 지정되지 않으면 <Content type="expression" value="'rct-text-light'"></Content>이,
darkStyle이 지정되지 않으면 <Content type="expression" value="'rct-text-dark'"></Content>가 기본 적용된다.  
<DefaultValue value="true" contents=""></DefaultValue>

### distance: `number{:js}`[#distance]
<Content type="expression" value="'pie', 'funnel'"></Content> 시리즈에서 label이 외부에 표시될 때 label과 시리즈 본체와의 기본 간격.  
<DefaultValue value="25" contents=""></DefaultValue>
### effect: `'background'|'none'|'outline'{:js}`[#effect]
- `'background'` 텍스트 배경 상자를 표시한다.      배경 상자에 [backgroundStyle](#backgroundStyle) 이 적용된다.  스타일이 적용되지 않으면 기본 <Content type="expression" value="'rct-text-background'"></Content>이 적용된다.
- `'none'` 효과 없음.
- `'outline'` 텍스트 색상과 대조되는 색상으로 텍스트 외곽을 구분 표시한다.

<DefaultValue value="'none'" contents=""></DefaultValue>
