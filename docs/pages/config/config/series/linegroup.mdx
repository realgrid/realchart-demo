## series[type=linegroup]

## Properties

### displayInLegend: `boolean{:typescript}`
명시적 false로 지정하면 legend에 표시하지 않는다.  
`default: true`  

### layout


`default: SeriesGroupLayout.DEFAULT`  

### layoutMax: `number{:typescript}`
layout 이  SeriesGroupLayout.FILL 일 때 상대적 최대값.

  
`default: 100`  

### style: `SVGStyles | string{:typescript}`
스타일셋 혹은 class selector.  

### xAxis: `string | number{:typescript}`

### yAxis: `string | number{:typescript}`

### visible: `boolean{:typescript}`
표시 여부.
모델에 따라 기본값이 다르다.
