## **series[type=pareto]**
Pareto 시리즈<br/>
참조하는 원본 시리즈의 누적 비율을 표시한다.<br/> [source](#source) 로 지정된 시리즈의 데이터포인트 값들로 부터 누적 포인트들을 계산해서 표시한다.
## Properties

### col: `number{:js}`[#col]
분할 모드일 때 시리즈가 표시될 pane의 수평 index.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### color: `string | number{:js}`[#color]
데이터 포인트 기본 색.<br/>
숫자로 지정하면 정수로 변환된 값에 해당하는 팔레트 색상으로 설정된다.
<Content type="expression" value="'var(--color-n)'"></Content>으로 지정한 것과 동일하며, 1 ~ 12 사이의 값으로 지정한다.<br/> [pointColors](#pointcolors) 나 [pointStyleCallback](#pointstylecallback) 으로 설정된 색상이 이 속성으로 설정한 색상보다 우선한다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### colorField: `string{:js}`[#colorField]
<Content type="keyword" value="undefined"></Content>이면, data point의 값이 객체일 때 <Content type="expression" value="'color'"></Content>.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### data: `any{:js}`[#data]
데이터포인터들을 생성하는 데 사용되는 값 목록.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### label: `string{:js}`[#label]
이 시리즈를 나타내는 텍스트.<br/>
레전드나 툴팁에서 시리즈를 대표한다.
이 속성이 지정되지 않으면 [name](#name) 이 사용된다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### [marker](./pareto/marker)
데이터 포인트 maker 설정 정보.  

### name: `string{:js}`[#name]
시리즈 이름.<br/>
시리즈 생성시 지정된 후 변경할 수 없다.
차트의 다른 구성 요소에서 이 시리즈를 참조할 때 사용되며,
레전드나 툴팁에서 시리즈를 나타내는 텍스트로도 사용된다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### noClip: `boolean{:js}`[#noClip]
<Content type="keyword" value="true"></Content>로 지정하면 body를 벗어난 data point 영역도 표시된다.<br/>
값을 지정하지 않으면 polar 차트에서는 <Content type="keyword" value="true"></Content>, 직교 차트에서는 <Content type="keyword" value="false"></Content>이다.
group에 포함되면 group의 noClip 설정을 따른다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### nullAsBase: `boolean{:js}`[#nullAsBase]
<Content type="keyword" value="null"></Content>인 y값을 [baseValue](#baseValue) 로 간주한다.  
<DefaultValue value="false" contents=""></DefaultValue>
### onPointClick[#onPointClick]
데이터 point가 클릭될 때 호출되는 이벤트 콜백.<br/>
명시적 <Content type="keyword" value="true"></Content>를 리턴하면 기본 동작이 진행되지 않는다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### onPointHover[#onPointHover]
마우스가 데이터 point 위에 올라가거나 빠져나갈 때 호출되는 이벤트 콜백.<br/>
빠져나가는 경우 args 매개변수는 <Content type="keyword" value="null"></Content>이다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### pointColors: `string | string[]{:js}`[#pointColors]
데이터 포인트별 색들을 지정한다.<br/>
색 배열로 지정하거나, <Content type="expression" value="'colors'"></Content> asset으로 등록된 이름을 지정할 수 있다.<br/> [pointStyleCallback](#pointstylecallback) 으로 설정된 색상이나 데이터포인트별로 지정한 색상이 이 속성으로 설정한 색상보다 우선한다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### [pointLabel](./pareto/pointLabel)
데이터포인트 label 설정 모델.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### pointStyle: `SVGStyles | string{:js}`[#pointStyle]
모든 데이터포인트에 적용되는 inline 스타일셋.<br/> [style](/config/config/series/pareto#style) 로 설정되는 시리즈의 inline 스타일이
데이터포인터에 적용되지 않는 경우 이 속성을 사용할 수 있다. [pointColors](#pointcolors) 나 [color](#color) 가 설정되면 이 속성으로 설정된 색상은 무시된다.
또, [pointStyleCallback](#pointstylecallback) 으로 설정된 스타일이 이 속성 스타일보다 우선한다.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fillOpacity | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| rx | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeDasharray | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |  
| textAlign | `'center' \| 'left' \| 'right'{:js}` | true |<DefaultValue value="undefined" contents=""></DefaultValue>
### pointStyleCallback[#pointStyleCallback]
데이터 point의 동적 스타일 콜백.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### row: `number{:js}`[#row]
분할 모드일 때 시리즈가 표시될 pane의 수직 index.  
<DefaultValue value="undefined" contents=""></DefaultValue>

| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| rx | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeDasharray | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |  
| textAlign | `'center' \| 'left' \| 'right'{:js}` | true |<DefaultValue value="undefined" contents=""></DefaultValue>
### tooltipText: `string{:js}`[#tooltipText]
데이터포인트 툴팁 텍스트.  
<DefaultValue value="'<b>${name}</b><br>${series}:<b>" contents="${yValue}</b>'"></DefaultValue>
### [trendline](./pareto/trendline)
추세선 설정 모델.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### viewRangeValue: `'x' | 'y' | 'z'{:js}`[#viewRangeValue]
ranges가 적용되는 값.<br/>
지정하지 않으면 시리즈 종류에 띠라 자동 적용된다.
<Content type="expression" value="'line' 시리즈 계열은 'x', 나머지는 'y'"></Content>가 된다.
현재 <Content type="expression" value="'z'"></Content>은 range는 bubble 시리즈에만 적용할 수 있다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### viewRanges: `IValueRanges | IValueRange[]{:js}`[#viewRanges]
값 범위 목록.<br/>
범위별로 다른 스타일을 적용할 수 있다.
범위들은 중첩될 수 없다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### visible: `boolean{:js}`[#visible]
표시 여부.  
<DefaultValue value="true" contents=""></DefaultValue>
### visibleInLegend: `boolean{:js}`[#visibleInLegend]
명시적 <Content type="keyword" value="false"></Content>로 지정하면 legend에 표시하지 않는다.  
<DefaultValue value="true" contents=""></DefaultValue>
### xAxis: `string | number{:js}`[#xAxis]
그룹에 포함되면 그룹 설정을 따른다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### xField: `string | number{:js}`[#xField]
json 객체나 배열로 전달되는 데이터포인트 정보에서 x 값을 지정하는 속성명이나 인덱스.<br/>
<Content type="keyword" value="undefined"></Content>이면, data point의 값이 array일 때는 0, 객체이면 <Content type="expression" value="'x'"></Content>.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### xStart: `any{:js}`[#xStart]
연결된 x축이 연속 축(카테고리축이 아닌)일 때, x축 값이 설정되지 않은 첫번째 데이터 point에 설정되는 x값.<br/>
이 후에는 [xStep](#xstep) 씩 증가시키면서 설정한다.
이 속성이 지정되지 않은 경우 [xStart](/config/config/options#xstart) 가 적용된다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### xStep: `string | number{:js}`[#xStep]
연결된 x축이 연속 축(카테고리축이 아닌)일 때, x축 값이 설정되지 않은 데이터 point에 지정되는 x값의 간격.<br/>
첫번째 값은 [xStart](#xstart) 로 설정한다.
time 축일 때, 정수 값 대신 시간 단위(<Content type="expression" value="'y', 'm', 'w', 'd', 'h', 'n', 's'"></Content>)로 지정할 수 있다.
이 속성이 지정되지 않으면 [xStep](/config/config/options#xstep) 이 적용된다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### yAxis: `string | number{:js}`[#yAxis]
그룹에 포함되면 그룹 설정을 따른다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### yField: `string | number | Function{:js}`[#yField]
json 객체나 배열로 전달되는 데이터포인트 정보에서 y 값을 지정하는 속성명이나 인덱스.<br/>
<Content type="keyword" value="undefined"></Content>이면, data point의 값이 array일 때는 1, 객체이면 <Content type="expression" value="'y'"></Content>.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### zField: `string | number | Function{:js}`[#zField]
json 객체나 배열로 전달되는 데이터포인트 정보에서 z 값을 지정하는 속성명이나 인덱스.<br/>
<Content type="keyword" value="undefined"></Content>이면, data point의 값이 array일 때는 2, 객체이면 <Content type="expression" value="'z'"></Content>.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### zOrder: `number{:js}`[#zOrder]
시리즈 표시 순서를 지정할 수 있다.<br/>
값이 클 수록 나중에(위에) 표시된다.  
<DefaultValue value="0" contents=""></DefaultValue>
