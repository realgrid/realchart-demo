## [marker](/config/config/marker)

## Properties

### firstVisible: `'default'|'hidden'|'visible'{:js}`[#firstVisible]
첫번째 point의 marker 표시 여부.  
- `'default'` visible 속성에 따른다.
- `'hidden'` visible 속성과 상관없이 항상 표시하지 않는다.
- `'visible'` visible 속성과 상관없이 항상 표시한다.

`default: 'default'`  

### lastVisible: `'default'|'hidden'|'visible'{:js}`[#lastVisible]
첫번째 point의 marker 표시 여부.  
- `'default'` visible 속성에 따른다.
- `'hidden'` visible 속성과 상관없이 항상 표시하지 않는다.
- `'visible'` visible 속성과 상관없이 항상 표시한다.

`default: 'default'`  

### maxVisible: `'default'|'hidden'|'visible'{:js}`[#maxVisible]
최대값 point들의 marker 표시 여부.  
- `'default'` visible 속성에 따른다.
- `'hidden'` visible 속성과 상관없이 항상 표시하지 않는다.
- `'visible'` visible 속성과 상관없이 항상 표시한다.

`default: 'default'`  

### minVisible: `'default'|'hidden'|'visible'{:js}`[#minVisible]
최소값 point들의 marker 표시 여부.  
- `'default'` visible 속성에 따른다.
- `'hidden'` visible 속성과 상관없이 항상 표시하지 않는다.
- `'visible'` visible 속성과 상관없이 항상 표시한다.

`default: 'default'`  

### radius: `number{:js}`[#radius]
`default: 4`  

### shape[#shape]
명시적으로 지정하지 않으면 typeIndex에 따라 Shapes 중 하나로 돌아가면서 설정된다.  



### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fillOpacity | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |
### visible: `boolean{:js}`[#visible]
표시 여부.  
`default: true`
