## series[type=areagroup]

## Properties

### displayInLegend: `boolean`
명시적 false로 지정하면 legend에 표시하지 않는다.  
`default: true`  

### layout
`default: SeriesGroupLayout.DEFAULT`  

### layoutMax: `number`
layout 이  SeriesGroupLayout.FILL 일 때 상대적 최대값.

  
`default: 100`  

### style
스타일셋 혹은 class selector.  

### xAxis: `string | number`

### yAxis: `string | number`
