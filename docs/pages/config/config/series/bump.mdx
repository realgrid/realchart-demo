## **series[type=bump]**
포함된 시리즈들의 y값들을 비교해서 순위로 시리즈를 표시한다.


포함된 시리즈들의 x값이 동일한 data point y값들을 비교해서 순위를 yValue로 재설정한다.
## Properties

### layout[#layout]


<DefaultValue value="SeriesGroupLayout.DEFAULT" contents=""></DefaultValue>
### layoutMax: `number{:js}`[#layoutMax]
[layout](#layout) 이 `fill` 일 때 상대적 최대값.  
<DefaultValue value="100" contents=""></DefaultValue>
### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fillOpacity | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| rx | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeDasharray | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |  
| textAlign | `'center' \| 'left' \| 'right'{:js}` | true |
### tooltipFooter: `string{:js}`[#tooltipFooter]
그룹 툴팁의 아래쪽에 표시되는 텍스트.  

### tooltipHeader: `string{:js}`[#tooltipHeader]
그룹 툴팁의 위쪽에 표시되는 텍스트.


tooltipHeader
tooltipRow,
tooltipRow,
...
tooltipFooter
형태로 툴팁이 표시된다.  
<DefaultValue value="'<b>${name}</b>'" contents=""></DefaultValue>
### tooltipRow: `string{:js}`[#tooltipRow]
그룹 툴팁에 각 시리즈별 표시되는 포인트 툴팁 텍스트.  
<DefaultValue value="'${series}:<b>" contents="${yValue}</b>'"></DefaultValue>
### visible: `boolean{:js}`[#visible]
표시 여부.  
<DefaultValue value="true" contents=""></DefaultValue>
### visibleInLegend: `boolean{:js}`[#visibleInLegend]
명시적 <Content type="keyword" value="false"></Content>로 지정하면 legend에 표시하지 않는다.  
<DefaultValue value="true" contents=""></DefaultValue>
### xAxis: `string | number{:js}`[#xAxis]
이 그룹에 포함된 시리즈들은 자신의 설정 대신 이 설정을 따른다.  

### yAxis: `string | number{:js}`[#yAxis]
이 그룹에 포함된 시리즈들은 자신의 설정 대신 이 설정을 따른다.
