## **marker**
데이터 포인트 maker 설정 정보.
## Properties

### firstVisible: `boolean{:js}`[#firstVisible]
첫번째 point의 marker 표시 여부.

<Content type="keyword" value="true"></Content>로 지정하면 모델 visible과 상관없이 표시하고,
<Content type="keyword" value="false"></Content>면 상관없이 표시하지 않는다.
아니면 모델의 visible을 따른다.  
<DefaultValue value="undefined" contents=""></DefaultValue>

### shape: `'circle'|'diamond'|'rectangle'|'square'|'star'|'triangle'{:js}`[#shape]
명시적으로 지정하지 않으면 typeIndex에 따라 Shapes 중 하나로 돌아가면서 설정된다.  
- `'circle'` 원
- `'diamond'` 다아이몬드
- `'rectangle'` 사각형
- `'square'` 정사각형
- `'star'` 별 모양
- `'triangle'` 삼각형

<DefaultValue value="undefined" contents=""></DefaultValue>
