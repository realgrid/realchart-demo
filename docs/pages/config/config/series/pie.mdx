## series[type=pie]

## Properties

### autoSlice: `boolean{:javascript}`[#autoSlice]
클릭한 데이터 포인트를 slice 시킨다.
기존 slice 됐던 포인트는 원복된다.  
`default: true`  

### borderRadius: `number{:javascript}`[#borderRadius]
`default: 0`  

### center: `string | number{:javascript}`[#center]

### centerX: `string | number{:javascript}`[#centerX]

### centerY: `string | number{:javascript}`[#centerY]

### clipped: `boolean{:javascript}`[#clipped]
body 영역을 벗어난 data point view는 잘라낸다.  
`default: false`  

### clockwise: `boolean{:javascript}`[#clockwise]
true면 시계 방향으로 회전한다.  
`default: true`  

### color: `string{:javascript}`[#color]
데이터 포인트 기본 색.  

### colorField: `string{:javascript}`[#colorField]
undefined이면, data point의 값이 객체일 때 'color'.  

### dataProp: `string{:javascript}`[#dataProp]
undefined이면 "data".  

### displayInLegend: `boolean{:javascript}`[#displayInLegend]
명시적 false로 지정하면 legend에 표시하지 않는다.  
`default: true`  

### groupSize: `number{:javascript}`[#groupSize]
`default: 1`  

### innerRadius: `string | number{:javascript}`[#innerRadius]
0보다 큰 값을 지정해서 도넛 형태로 표시할 수 있다.
시리즈 원호의 반지름에 대한 상대적 크기나 픽셀 수로 지정할 수 있다.
 innerText 로 도넛 내부에 표시될 텍스트를 지정할 수 있다.  

### innerText[#innerText]
innerRadius 가 0보다 클 때, 도넛 내부에 표시되는 텍스트.
기본 클래스 selector는 <b>'rct-pie-series-inner'</b>이다.  
`default: ...`  

### labelDistance: `number{:javascript}`[#labelDistance]
`default: 25`  

### legendByPoint: `boolean{:javascript}`[#legendByPoint]
데이터 포인트별 legend 항목을 표시한다.  
`default: false`  

### onPointClick[#onPointClick]
데이터 point가 클릭될 때 발생하는 이벤트 콜백.
명시적 true를 리턴하면 기본 동작이 진행되지 않는다.  

### pointColors: `boolean | string[]{:javascript}`[#pointColors]
데이터 포인트별 색들을 지정한다.
false로 지정하면 모든 포인트들이 시리즈 색으로 표시된다.
true로 지정하면 기본 색들로 표시된다.
색 문자열 배열로 지정하면 포함된 색 순서대로 표시된다.
undefined나 null이면 시리즈 종류에 따라 false 혹은 true로 해석된다.  

### pointStyleCallback[#pointStyleCallback]
데이터 point의 동적 스타일 콜백.  

### radius: `string | number{:javascript}`[#radius]
시리즈 원호의 반지름.
픽셀 크기나 plot 영역 크기에 대한 상대적 크기로 지정할 수 있다.
'50%'로 지정하면 plot 영역의 width나 height중 작은 크기와 동일한 반지름으로 표시된다.  
`default: RadialSeries.DEF_RADIUS`  

### sliceDuration: `number{:javascript}`[#sliceDuration]
Slice animation duration.
밀리세컨드(ms) 단위로 지정.  
`default: 300`  

### sliceOffset: `string | number{:javascript}`[#sliceOffset]
`default: '7%'`  

### startAngle: `number{:javascript}`[#startAngle]
시리즈 원호 시작 각도.
지정하지 않거나 잘못된 값이면 0으로 계산된다.
0은 시계의 12시 위치다.  
`default: 0`  

### style: `SVGStyles | string{:javascript}`[#style]
스타일셋 혹은 class selector.  

### totalAngle: `number{:javascript}`[#totalAngle]
시리즈 원호 전체 각도.
0 ~ 360 사이의 값으로 지정해야 한다.
범위를 벗어난 값은 범위 안으로 조정된다.
지정하지 않거나 잘못된 값이면 360으로 계산된다.  
`default: 360`  

### xAxis: `string | number{:javascript}`[#xAxis]

### xField: `string | number{:javascript}`[#xField]
undefined이면, data point의 값이 array일 때는 0, 객체이면 'x'.  

### xStart: `any{:javascript}`[#xStart]
x축 값이 설정되지 않은 첫번째 데이터 point에 설정되는 x값.
이 후에는  xStep 씩 증가시키면서 설정한다.
이 속성이 지징되지 않은 경우  ChartOptions.xStart 가 적용된다.  

### xStep: `string | number{:javascript}`[#xStep]
x축 값이 설정되지 않은 데이터 point에 지정되는 x값의 간격.
첫번째 값은  xStart 로 설정한다.
time 축일 때, 정수 값 대신 시간 단위('y', 'm', 'd', 'h', 'n', 's')로 지정할 수 있다.
이 속성이 지정되지 않으면  ChartOptions.xStep 이 적용된다.  

### yAxis: `string | number{:javascript}`[#yAxis]

### yField: `string | number{:javascript}`[#yField]
undefined이면, data point의 값이 array일 때는 1, 객체이면 'y'.  

### zOrder: `number{:javascript}`[#zOrder]
`default: 0`  

### visible: `boolean{:javascript}`[#visible]
표시 여부.\
모델에 따라 기본값이 다르다.
