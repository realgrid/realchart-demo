## series[type=boxplot]

## Properties

### col: `number{:js}`[#col]
분할 모드일 때 시리즈가 표시될 pane의 수직 위치.  

### color: `string{:js}`[#color]
데이터 포인트 기본 색.  

### colorField: `string{:js}`[#colorField]
undefined이면, data point의 값이 객체일 때 'color'.  

### dataProp: `string{:js}`[#dataProp]
undefined이면 "data".  

### highField: `string{:js}`[#highField]

### label: `string{:js}`[#label]
이 시리즈를 나타내는 텍스트.\
레전드나 툴팁에서 시리즈를 대표한다.
이 속성이 지정되지 않으면  **[name](#name)** 이 사용된다.  

### lowField: `string{:js}`[#lowField]

### midField: `string{:js}`[#midField]

### minField: `string{:js}`[#minField]

### name: `string{:js}`[#name]
시리즈 이름.\
시리즈 생성시 지정되고 변경할 수 없다.  

### noClip: `boolean{:js}`[#noClip]
true로 지정하면 body를 벗어난 data point 영역도 표시된다.  
<DefaultValue value="false" contents=""></DefaultValue>
### onPointClick[#onPointClick]
데이터 point가 클릭될 때 발생하는 이벤트 콜백.
명시적 true를 리턴하면 기본 동작이 진행되지 않는다.  

### pointColors: `boolean | string[]{:js}`[#pointColors]
데이터 포인트별 색들을 지정한다.\
false로 지정하면 모든 포인트들이 시리즈 색으로 표시된다.
true로 지정하면 기본 색들로 표시된다.
색 문자열 배열로 지정하면 포함된 색 순서대로 표시된다.
undefined나 null이면 시리즈 종류에 따라 false 혹은 true로 해석된다.  

### [pointLabel](./boxplot/pointLabel)
Series data point label options.  

### pointPadding: `number{:js}`[#pointPadding]
이 시리즈의 point가 차지하는 영역 중에서 point bar 양쪽 끝에 채워지는 빈 영역의 크기.


point가 차지할 원래 크기에 대한 상대 값으로서,
0 ~ 1 사이의 비율 값으로 지정한다.  
<DefaultValue value="undefined" contents="한 카테고리에 cluster 가능한 시리즈가 하나만 표시되면 0.25, group에 포함된 경우 0.1, 여러 시리즈와 같이 표시되면 0.2."></DefaultValue>
### pointStyleCallback[#pointStyleCallback]
데이터 point의 동적 스타일 콜백.  

### pointWidth: `number{:js}`[#pointWidth]
시리즈가 그룹에 포함되지 않거나,
포함된 그룹의 layout이  `default` 이거나 특별히 설정되지 않아서,
그룹에 포함된 시리즈들의 data point가 옆으로 나열되어 표시될 때,
포인트 표시 영역 내에서 이 시리즈의 포인트가 차지하는 영역의 상대 크기.
예를 들어 이 시리즈의 속성값이 1이고 다른 시리즈의 값이 2이면 다른 시리즈의 data point가 두 배 두껍게 표시된다.
또, 그룹에 포함되고 그룹의 layout이  `default`  
<DefaultValue value="1" contents=""></DefaultValue>
### row: `number{:js}`[#row]
분할 모드일 때 시리즈가 표시될 pane의 수평 위치.  

### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fillOpacity | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeDasharray | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |
### [tooltip](./boxplot/tooltip)
Series.tooltip 모델.  

### [trendline](./boxplot/trendline)
시리즈 추세선.  

### viewRangeValue: `'z' | 'y' | 'x'{:js}`[#viewRangeValue]
ranges가 적용되는 값.\
지정하지 않으면 시리즈 종류에 띠라 자동 적용된다.
'line' 시리즈 계열은 'x', 나머지는 'y'가 된다.
현재 'z'은 range는 bubble 시리즈에만 적용할 수 있다.  

### viewRanges: `IValueRanges | IValueRange[]{:js}`[#viewRanges]
값 범위 목록.\
범위별로 다른 스타일을 적용할 수 있다.
범위들은 중첩될 수 없다.  

### visible: `boolean{:js}`[#visible]
표시 여부.  
<DefaultValue value="true" contents=""></DefaultValue>
### visibleInLegend: `boolean{:js}`[#visibleInLegend]
명시적 false로 지정하면 legend에 표시하지 않는다.  
<DefaultValue value="true" contents=""></DefaultValue>
### xAxis: `string | number{:js}`[#xAxis]

### xField: `string | number{:js}`[#xField]
undefined이면, data point의 값이 array일 때는 0, 객체이면 'x'.  

### xStart: `any{:js}`[#xStart]
x축 값이 설정되지 않은 첫번째 데이터 point에 설정되는 x값.\
이 후에는  **[xStep](#xstep)** 씩 증가시키면서 설정한다.
이 속성이 지징되지 않은 경우  [xStart](/config/config/options#xstart) 가 적용된다.  

### xStep: `string | number{:js}`[#xStep]
x축 값이 설정되지 않은 데이터 point에 지정되는 x값의 간격.\
첫번째 값은  **[xStart](#xstart)** 로 설정한다.
time 축일 때, 정수 값 대신 시간 단위('y', 'm', 'w', 'd', 'h', 'n', 's')로 지정할 수 있다.
이 속성이 지정되지 않으면  [xStep](/config/config/options#xstep) 이 적용된다.  

### yAxis: `string | number{:js}`[#yAxis]

### yField: `string | number{:js}`[#yField]
undefined이면, data point의 값이 array일 때는 1, 객체이면 'y'.  

### zOrder: `number{:js}`[#zOrder]
<DefaultValue value="0" contents=""></DefaultValue>
