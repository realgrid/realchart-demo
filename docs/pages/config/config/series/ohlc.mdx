## **series[type=ohlc]**
Ohlc 시리즈.<br/>
시가-고가-저가-종가 차트.
시간 경과에 따른 가격의 움직임을 설명하는 데 사용된다.<br/><br/> [data](#data) 는 아래 형식들로 전달할 수 있다.
[주의] 데이터포인트 구성에 필요한 모든 값을 제공하지 않으면 <Content type="keyword" value="null"></Content>이 된다.
[주의] high와 y값은 동일한 값이다.

###### 단일 값 및 값 배열
|형식|설명|
|---|---|
|y|단일 숫자면 low, y값. x 값은 순서에 따라 자동 결정.|
|[]|빈 배열이면 <Content type="keyword" value="null"></Content>. x 값은 순서에 따라 자동 결정.|
|[min, low, mid, high | y]|형식 설명 순서대로 값 결정. x 값은 데이터포인트 순서에 따라 자동 결정.|
|[x, min, low, mid, high | y]|형깃 설명 순서대로 값 결정.<br/> 또는 [xField](#xfield) 속성이 숫자이면 x값, [minField](#minField) 는 min값,<br/> [lowField](#lowfield) 는 low값, [midField](#midField) 는 mid값,<br/> [highField](#highfield) 는 high값, [yField](#yfield) 는 y값의 index.|

###### json 배열
|Series 속성|설명|
|---|---|
| [xField](#xfield) |속성 값, 또는 <Content type="expression" value="'x', 'name', 'label'"></Content> 속성들 중 순서대로 값이 설정된 것이 x 값이 된다.|
| [lowField](#lowfield) |속성 값, 또는 <Content type="expression" value="'low'"></Content> 속성 값이 low 값이 된다.|
| [openField](#openfield) |속성 값, 또는 <Content type="expression" value="'open'"></Content> 속성 값이 open 값이 된다.|
| [closeField](#closefield) |속성 값, 또는 <Content type="expression" value="'mid'"></Content> 속성 값이 close 값이 된다.|
| [highField](#highfield) |속성 값, 또는 <Content type="expression" value="'high'"></Content> 속성 값이 high 값이 된다. 지정하지 않으면 yField 값이 사용된다.|
| [yField](#yfield) |속성 값, 또는 <Content type="expression" value="'y', 'value'"></Content> 속성들 중 순서대로 값이 설정된 것이 y 값이 된다.|
| [colorField](#colorfield) |속성 값, 또는 <Content type="expression" value="'color'"></Content> 속성 값으로 데이터포인트의 개별 색상으로 지정된다.|

### pointWidth: `number{:js}`[#pointWidth]
시리즈가 그룹에 포함되지 않거나,
포함된 그룹의 layout이 `default` 이거나 특별히 설정되지 않아서,
그룹에 포함된 시리즈들의 data point가 옆으로 나열되어 표시될 때,
포인트 표시 영역 내에서 이 시리즈의 포인트가 차지하는 영역의 상대 크기.
예를 들어 이 시리즈의 속성값이 1이고 다른 시리즈의 값이 2이면 다른 시리즈의 data point가 두 배 두껍게 표시된다.
또, 그룹에 포함되고 그룹의 layout이 `default`  
<DefaultValue value="1" contents=""></DefaultValue>
