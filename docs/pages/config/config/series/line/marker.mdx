## [marker](/config/config/marker)

## Properties

### firstVisible: `'default'|'hidden'|'visible'{:js}`[#firstVisible]
첫번째 point의 marker 표시 여부.  
- `'default'` visible 속성에 따른다.
- `'hidden'` visible 속성과 상관없이 항상 표시하지 않는다.
- `'visible'` visible 속성과 상관없이 항상 표시한다.

<DefaultValue value="'default'" contents=""></DefaultValue>
