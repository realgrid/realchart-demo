## annotation[type=image]
이미지 Annotation 모델.
## Properties

### align[#align]
수평 배치.  


`default: Align.LEFT`  

### front: `boolean{:js}`[#front]
true로 지정하면 시리즈들 위에 표시된다.  
`default: false`  

### imageUrl: `string{:js}`[#imageUrl]
이미지 경로.  

### name: `string{:js}`[#name]
Annotation 이름.
동적으로 Annotation을 다루기 위해서는 반드시 지정해야 한다.  

### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fillOpacity | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |
### height: `string | number{:js}`[#height]
Annotation 높이.
픽셀 단위의 고정 값이나, plot 영역에 대한 상태 크기롤 지정할 수 있다.  

### visible: `boolean{:js}`[#visible]
표시 여부.\  
`default: true`  

### width: `string | number{:js}`[#width]
Annotation 너비.
픽셀 단위의 고정 값이나, plot 영역에 대한 상태 크기롤 지정할 수 있다.
