| rx | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeDasharray | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |  
| textAlign | `'center' \| 'left' \| 'right'{:js}` | true |
### front: `boolean{:js}`[#front]
<Content type="keyword" value="true"></Content>로 지정하면 시리즈들 위에 표시된다.  
<DefaultValue value="false" contents=""></DefaultValue>
### name: `string{:js}`[#name]
Annotation 이름.

동적으로 Annotation을 다루기 위해서는 반드시 지정해야 한다.  

### numberFormat: `string{:js}`[#numberFormat]
[text](#text) 에 동적으로 전달되는 값이 숫자일 때 사용되는 표시 형식.  

### rotation: `number{:js}`[#rotation]
회전 각도.

0 ~ 360 사이의 값으로 지정한다.  

### scope[#scope]
차트 모델에서 지정된 annotationd의 표시 기준 영역.
