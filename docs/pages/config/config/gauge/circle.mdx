## gauge[type=circle]
게이지 모델.
## Properties

### animatable: `boolean`
value  변화를 애니메이션으로 표현한다.  
`default: true`  

### bottom
position 이  LegendPosition.PLOT plot 일 때, plot 영역의 하단 모서리와 legend의 간격.
 top 이 지정되면 이 속성은 무시된다.  

### centerX
게이지 중심 수평 위치.
픽셀 단위의 크기나, plot 영역 전체 너비에 대한 상대적 크기로 지정할 수 있다.  
`default: CircularGauge.DEF_CENTER`  

### centerY
게이지 중심 수직 위치.
픽셀 단위의 크기나, plot 영역 전체 높이에 대한 상대적 크기로 지정할 수 있다.  
`default: CircularGauge.DEF_CENTER`  

### clockwise: `boolean`
true면 시계 방향으로 회전한다.  
`default: true`  

### duration: `number`
Animation duration.  
`default: 500`  

### hand
게이지 바늘 설정 모델.  
`default: ...`  

### height
게이지 높이.
픽셀 단위의 고정 값이나, plot 영역에 대한 상태 크기롤 지정할 수 있다.  

### innerRadius
내부 원호의 반지름.
픽셀 단위의 크기나,  radius 에 대한 상대적 크기로 지정할 수 있다.
'100%'이면 게이지 원호의 반지름과 동일하다.  
`default: CircularGauge.DEF_INNER`  

### innerStyle
내부 원에 적용할 스타일셋 혹은 class selector.  

### label
게이지 중앙에 표시되는 label 설정 모델  

### left: `any`
`default: ...`  

### maxValue: `number`
최대값.  
`default: 100`  

### minValue: `number`
최소값.  
`default: 0`  

### name: `string`
게이지 이름.
동적으로 게이지를 다루기 위해서는 반드시 지정해야 한다.  

### pin
게이지 중앙에 표시되는 핀 설정 모델.  
`default: ...`  

### radius
게이지 원호의 반지름.
픽셀 단위의 크기나, plot 영역 전체 크기(너비와 높이 중 작은 값)에 대한 상대적 크기로 지정할 수 있다.
'50%'로 지정하면 plot 영역의 width나 height중 작은 크기와 동일한 반지름으로 표시된다.  
`default: CircularGauge.DEF_RADIUS`  

### right
position 이  LegendPosition.PLOT plot 일 때, plot 영역의 우측 모서리와 legend의 간격.
 left 가 지정되면 이 속성은 무시된다.  

### rim
게이지 배경 원호 테두리 설정 모델.  
`default: ...`  

### size
width 와  height 를 동시에 설정한다.  
`default: '100%'`  

### startAngle: `number`
게이지 원호 시작 각도.
지정하지 않거나 잘못된 값이면 0으로 계산된다.
0은 시계의 12시 위치다.  
`default: 0`  

### style
스타일셋 혹은 class selector.  

### sweepAngle: `number`
게이지 원호 전체 각도.
0 ~ 360 사이의 값으로 지정해야 한다.
범위를 벗어난 값은 범위 안으로 조정된다.
지정하지 않거나 잘못된 값이면 360으로 계산된다.
예) 180 이면 반 원호가 된다.  
`default: 360`  

### top: `any`
`default: ...`  

### value: `number`
현재값.  
`default: 0`  

### valueRim
게이지 값 원호 테두리 설정 모델.  
`default: ...`  

### width
게이지 너비.
픽셀 단위의 고정 값이나, plot 영역에 대한 상태 크기롤 지정할 수 있다.
