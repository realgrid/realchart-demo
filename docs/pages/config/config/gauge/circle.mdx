## gauge[type=circle]
게이지 모델.
## Properties

### animatable: `boolean`
value  변화를 애니메이션으로 표현한다.  
`default: true`  

### bandRim
게이지 본체 주변이나 내부에 값 영역들을 구분해서 표시하는 band의 모델.  
`default: ...`  

### clockwise: `boolean`
true면 시계 방향으로 회전한다.  
`default: true`  

### duration: `number`
Animation duration.  
`default: 500`  

### hand
게이지 바늘 설정 모델.  
`default: ...`  

### innerStyle
내부 원에 적용할 스타일셋 혹은 class selector.  

### label
게이지 중앙에 표시되는 label 설정 모델  

### maxValue: `number`
최대값.
group에 포함되면 이 값 대신 group의 최대값이 사용된다.  
`default: 100`  

### minValue: `number`
최소값.
group에 포함되면 이 값 대신 group의 최소값이 사용된다.  
`default: 0`  

### name: `string`
게이지 이름.
동적으로 게이지를 다루기 위해서는 반드시 지정해야 한다.  

### pin
게이지 중앙에 표시되는 핀 설정 모델.  
`default: ...`  

### rim
게이지 배경 원호 테두리 설정 모델.  
`default: ...`  

### scaleRim
스케일 모델.  
`default: ...`  

### startAngle: `number`
게이지 원호 시작 각도.
지정하지 않거나 잘못된 값이면 0으로 계산된다.
0은 시계의 12시 위치다.  
`default: 0`  

### style
스타일셋 혹은 class selector.  

### sweepAngle: `number`
게이지 원호 전체 각도.
0 ~ 360 사이의 값으로 지정해야 한다.
범위를 벗어난 값은 범위 안으로 조정된다.
지정하지 않거나 잘못된 값이면 360으로 계산된다.
예) 180 이면 반 원호가 된다.  
`default: 360`  

### value: `number`
현재값.  
`default: 0`  

### valueRim
게이지의 값 원호 테두리 설정 모델.  
`default: ...`
