## secondHand
초침 설정 모델.
## Properties

### animatable: `boolean{:typescript}`
true면 초 이동 시 애니메이션 효과로 표시한다.  
`default: false`  

### duration: `number{:typescript}`
animatable 이 true일 때 애니메이션 기간.
밀리초 단위로 지정한다.  
`default: 200`  

### style
스타일셋 혹은 class selector.  

### thickness: `number{:typescript}`
픽셀 단위 바늘 두께.
침 종류에 따라 기본값이 다르다.  

### length
바늘 길이.
픽셀 단위로 직접 지정하거나 시계 반지름에 대한 상대 크기로 지정할 수 있다.
침 종류에 따라 기본값이 다르다.  

### visible: `boolean{:typescript}`
표시 여부.
모델에 따라 기본값이 다르다.
