## Properties

### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  

### vertical: `boolean{:js}`[#vertical]
true면 수직 bar로 표시한다.  
`default: false`  

### height: `string | number{:js}`[#height]
높이  

### ranges: `IValueRange[]{:js}`[#ranges]
값 범위 목록.
범위별로 다른 스타일을 적용할 수 있다.
