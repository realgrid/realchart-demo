## [label](/config/config/label)
label 설정 모델.
## Properties

### animatable: `boolean{:js}`[#animatable]
게이지 값 변경 애니메이션이 실행될 때, label도 따라서 변경시킨다.  
<DefaultValue value="true" contents=""></DefaultValue>
### autoContrast: `boolean{:js}`[#autoContrast]
텍스트가 data point 내부에 표시되는 경우 포인트 색상과 대조되도록 표시한다.


밝게 표시할 때는  [lightStyle](#lightstyle) 을 적용하고,
어둡게 표시할 때는  [darkStyle](#darkstyle) 이 적용된다.
brightStyle이 지정되지 않으면 <Content type="expression" value="'rct-text-light'"></Content>이,
darkStyle이 지정되지 않으면 <Content type="expression" value="'rct-text-dark'"></Content>가 기본 적용된다.  
<DefaultValue value="true" contents=""></DefaultValue>

### effect[#effect]

### gap: `string | number{:js}`[#gap]
label과 본체 사이의 간격.
픽셀값이나 게이지 전체 크기(position에 따라 높이나 너비)에 대한 상대값으로 지정할 수 있다.  

### height: `string | number{:js}`[#height]
position이 <Content type="expression" value="'top', 'bottom'"></Content>일 때, label 표시 영역 높이.
픽셀값이나 게이지 전체 높이에 대한 상대값으로 지정할 수 있다.

### maxHeight: `string | number{:js}`[#maxHeight]
position이 <Content type="expression" value="'top', 'bottom'"></Content>일 때, label 표시 영역 최대 높이.
픽셀값이나 게이지 전체 높이에 대한 상대값으로 지정할 수 있다.
 [height](#height) 가 지정되면 이 속성은 무시된다.  

### maxWidth: `string | number{:js}`[#maxWidth]
position이 <Content type="expression" value="'left', 'right'"></Content>일 때, label 표시 영역 최대 너비.
픽셀값이나 게이지 전체 너비에 대한 상대값으로 지정할 수 있다.
 [width](#width) 가 지정되면 이 속성은 무시된다.  

### numberFormat: `string{:js}`[#numberFormat]
label이 숫자일 때 표시 형식.  

### numberSymbols: `string{:js}`[#numberSymbols]
축의 tick 간격이 1000 이상인 큰 수를 표시할 때 
이 속성에 지정한 symbol을 이용해서 축약형으로 표시한다.  

### position: `'left' | 'top' | 'bottom' | 'right'{:js}`[#position]
값을 지정하지 않으면 게이지 표시 방향에 따라 자동으로 위치를 결정한다.  

### prefix: `string{:js}`[#prefix]
label 문자열 앞에 추가되는 문자열.
