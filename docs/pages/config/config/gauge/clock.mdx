## gauge[type=clock]
시계 게이지 모델.
## Properties

### centerX: `string | number{:javascript}`[#centerX]
게이지 중심 수평 위치.
픽셀 단위의 크기나, plot 영역 전체 너비에 대한 상대적 크기로 지정할 수 있다.  
`default: CircularGauge.DEF_CENTER`  

### centerY: `string | number{:javascript}`[#centerY]
게이지 중심 수직 위치.
픽셀 단위의 크기나, plot 영역 전체 높이에 대한 상대적 크기로 지정할 수 있다.  
`default: CircularGauge.DEF_CENTER`  

### duration: `number{:javascript}`[#duration]
Animation duration.  
`default: 500`  

### [hourHand](./clock/hourHand)
RealReport-Chart 라이브러리 클래스 모델의 최상위 base 클래스.

  

### [label](./clock/label)
RealReport-Chart 라이브러리 클래스 모델의 최상위 base 클래스.

  

### [minorTick](./clock/minorTick)
RealReport-Chart 라이브러리 클래스 모델의 최상위 base 클래스.

  

### [minuteHand](./clock/minuteHand)
RealReport-Chart 라이브러리 클래스 모델의 최상위 base 클래스.

  

### name: `string{:javascript}`[#name]
게이지 이름.
동적으로 게이지를 다루기 위해서는 반드시 지정해야 한다.  

### [pin](./clock/pin)
RealReport-Chart 라이브러리 클래스 모델의 최상위 base 클래스.

  

### radius: `string | number{:javascript}`[#radius]
게이지 원호의 반지름.
픽셀 단위의 크기나, plot 영역 전체 크기(너비와 높이 중 작은 값)에 대한 상대적 크기로 지정할 수 있다.
'50%'로 지정하면 plot 영역의 width나 height중 작은 크기와 동일한 반지름으로 표시된다.  
`default: CircularGauge.DEF_RADIUS`  

### [rim](./clock/rim)
RealReport-Chart 라이브러리 클래스 모델의 최상위 base 클래스.

  

### [secondHand](./clock/secondHand)
RealReport-Chart 라이브러리 클래스 모델의 최상위 base 클래스.

  

### style: `SVGStyles | string{:javascript}`[#style]
스타일셋 혹은 class selector.  

### [tick](./clock/tick)
RealReport-Chart 라이브러리 클래스 모델의 최상위 base 클래스.

  

### [tickLabel](./clock/tickLabel)
RealReport-Chart 라이브러리 클래스 모델의 최상위 base 클래스.

  

### time: `string | number | Date{:javascript}`[#time]
이 속성에 명시적으로 시간을 지정하면 현재 시간 대신 이 시각을 표시한다.  

### timezone: `number{:javascript}`[#timezone]
분단위로 시계에 표시할 지역의 timezone을 지정한다.
예) 뉴욕: -4 * 60, 파리: 2 * 60  

### active: `boolean{:javascript}`[#active]
시계 동작 여부.  
`default: true`  

### bottom: `string | number{:javascript}`[#bottom]
plot 영역의 아래쪽 모서리와 widget 사이의 간격.  

### height: `string | number{:javascript}`[#height]
게이지 높이.
픽셀 단위의 고정 값이나, plot 영역에 대한 상태 크기롤 지정할 수 있다.  

### left: `string | number{:javascript}`[#left]
plot 영역의 왼쪽 모서리와 widget 사이의 간격.  

### right: `string | number{:javascript}`[#right]
plot 영역의 오른쪽 모서리와 widget 사이의 간격.  

### size: `string | number{:javascript}`[#size]
width 와  height 를 동시에 설정한다.  

### top: `string | number{:javascript}`[#top]
plot 영역의 위쪽 모서리와 widget 사이의 간격.  

### visible: `boolean{:javascript}`[#visible]
표시 여부.\
모델에 따라 기본값이 다르다.
- [Fiddle](https:/jsfiddle.net/gh/get/library/pure/realgrid/realchart-demo/tree/master/bargroup/bargroup-multi)  

### width: `string | number{:javascript}`[#width]
게이지 너비.
픽셀 단위의 고정 값이나, plot 영역에 대한 상태 크기롤 지정할 수 있다.
