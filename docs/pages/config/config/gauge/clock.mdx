## gauge[type=clock]
시계 게이지 모델.
## Properties

### centerX
게이지 중심 수평 위치.
픽셀 단위의 크기나, plot 영역 전체 너비에 대한 상대적 크기로 지정할 수 있다.  
`default: CircularGauge.DEF_CENTER`  

### centerY
게이지 중심 수직 위치.
픽셀 단위의 크기나, plot 영역 전체 높이에 대한 상대적 크기로 지정할 수 있다.  
`default: CircularGauge.DEF_CENTER`  

### duration: `number`
Animation duration.  
`default: 500`  

### hourHand
시침 설정 모델.  
`default: ...`  

### label
label  
`default: ...`  

### minorTick
minor tick  
`default: ...`  

### minuteHand
분침 설정 모델.  
`default: ...`  

### name: `string`
게이지 이름.
동적으로 게이지를 다루기 위해서는 반드시 지정해야 한다.  

### pin
pin  
`default: ...`  

### radius
게이지 원호의 반지름.
픽셀 단위의 크기나, plot 영역 전체 크기(너비와 높이 중 작은 값)에 대한 상대적 크기로 지정할 수 있다.
'50%'로 지정하면 plot 영역의 width나 height중 작은 크기와 동일한 반지름으로 표시된다.  
`default: CircularGauge.DEF_RADIUS`  

### rim
rim 설정 모델.  
`default: ...`  

### secondHand
초침 설정 모델.  
`default: ...`  

### style
스타일셋 혹은 class selector.  

### tick
main tick.  
`default: ...`  

### tickLabel
tick label  
`default: ...`  

### time: `string | number | `
이 속성에 명시적으로 시간을 지정하면 현재 시간 대신 이 시각을 표시한다.  

### timezone: `number`
분단위로 시계에 표시할 지역의 timezone을 지정한다.
예) 뉴욕: -4 * 60, 파리: 2 * 60
