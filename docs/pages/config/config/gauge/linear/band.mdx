## band
게이지 본체 주변이나 내부에 값 영역들을 구분해서 표시하는 band의 모델.
## Properties

### position
표시 위치.  
`default: GaugeItemPosition.DEFAULT`  

### rangeLabel
position 이 'inside'일 때만 표시될 수 있다.  

### style
스타일셋 혹은 class selector.  

### thickness: `number`
밴드의 두께를 픽셀 단위로 지정한다.
 position 이 'inside'이면 이 속성은 무시되고, 게이지 본체의 두께와 동일하게 표시된다.  
`default: 7`  

### tickLabel
각 range의 양 끝에 해당하는 값을 표시한다.
