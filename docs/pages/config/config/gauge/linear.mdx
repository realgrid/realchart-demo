## **gauge[type=linear]**
선형 게이지 모델.<br/>
현재 값을 선형 막대로 표시한다.
또, [band](#band) 를 같이 표시하면, 여러 값 범위 중 어디에 속한 상태인 지를 나타낼 수 있다.
## Properties

### animatable: `boolean{:js}`[#animatable]
[value](#value) 변화를 애니메이션으로 표현한다.  
<DefaultValue value="true" contents=""></DefaultValue>
### [band](./linear/band)
게이지 본체 주변이나 내부에 값 영역들을 구분해서 표시하는 band의 모델.  
<DefaultValue value="..." contents=""></DefaultValue>
### bottom: `string | number{:js}`[#bottom]
plot 영역의 아래쪽 모서리와 widget 사이의 간격.<br/> [bottom](#bottom) 이 설정되면 이 속성은 무시된다.  
<DefaultValue value="undefined" contents=""></DefaultValue>
### col: `number{:js}`[#col]
분할 모드일 때 게이지가 표시될 pane의 수평 index.  
<DefaultValue value="undefined" contents=""></DefaultValue>
