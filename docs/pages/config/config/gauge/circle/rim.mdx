## rim
게이지 배경 원호 테두리 설정 모델.
## Properties

### style: `SVGStyles | string{:javascript}`[#style]
스타일셋 혹은 class selector.  

### ranges: `IValueRange[]{:javascript}`[#ranges]
값 범위 목록.
범위별로 다른 스타일을 적용할 수 있다.  

### segmentThickness: `string | number{:javascript}`[#segmentThickness]
segment 두께.
픽셀 단위 크기나, 게이지 원호 두께에 대한 상대 크기로 지정할 수 있다.  

### visible: `boolean{:javascript}`[#visible]
표시 여부.
모델에 따라 기본값이 다르다.
