## rim
게이지 배경 원호 테두리 설정 모델.
## Properties

### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |
### ranges: `IValueRange[]{:js}`[#ranges]
값 범위 목록.
범위별로 다른 스타일을 적용할 수 있다.  
- `IValueRange{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| color | `string{:js}` | false |  
| fromValue | `number{:js}` | true |  
| label | `string{:js}` | true |  
| style | `SVGStyles | string{:js}` | true |  
| toValue | `number{:js}` | true |
### segmentThickness: `string | number{:js}`[#segmentThickness]
segment 두께.
픽셀 단위 크기나, 게이지 원호 두께에 대한 상대 크기로 지정할 수 있다.  

### visible: `boolean{:js}`[#visible]
표시 여부.\  
`default: true`
