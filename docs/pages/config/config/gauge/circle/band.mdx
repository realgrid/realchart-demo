## band
게이지 본체 주변이나 내부에 값 영역들을 구분해서 표시하는 band의 모델.
## Properties

### position[#position]
표시 위치.  


`default: GaugeItemPosition.DEFAULT`  

### [rangeLabel](./circle/rangeLabel)
  

### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.  
- `SVGStyles{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| fill | `string{:js}` | true |  
| fontFamily | `string{:js}` | true |  
| fontSize | `string{:js}` | true |  
| fontStyle | `string{:js}` | true |  
| fontWeight | `string{:js}` | true |  
| stroke | `string{:js}` | true |  
| strokeWidth | `string{:js}` | true |
### [tickLabel](./circle/tickLabel)
RealReport-Chart 라이브러리 클래스 모델의 최상위 base 클래스.  

### ranges: `IValueRange[]{:js}`[#ranges]
값 범위 목록.
범위별로 다른 스타일을 적용할 수 있다.  
- `IValueRange{:js}` 

| Name | Type | Optional |  
| ----- | ----- | ----- |  
| color | `string{:js}` | false |  
| fromValue | `number{:js}` | true |  
| label | `string{:js}` | true |  
| style | `SVGStyles | string{:js}` | true |  
| toValue | `number{:js}` | true |
### thickness: `string | number{:js}`[#thickness]
밴드의 두께를 픽셀 단위나, 
게이지 rim의 두께에 대한 상대적 크기로 지정할 수 있다.\
값을 지정하지 않으면 게이지 본체 내부에 표시될 때 '100%'(본체 두께와 동일)로 표시되고,
외부에 있을 때는 7 픽셀로 두께로 표시된다.  

### visible: `boolean{:js}`[#visible]
표시 여부.\  
`default: true`
