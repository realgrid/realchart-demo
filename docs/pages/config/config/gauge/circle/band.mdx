## band
게이지 본체 주변이나 내부에 값 영역들을 구분해서 표시하는 band의 모델.
## Properties

### position
표시 위치.  


`default: GaugeItemPosition.DEFAULT`  

### [rangeLabel](./circle/rangeLabel)
  

### style: `SVGStyles | string{:typescript}`
스타일셋 혹은 class selector.  

### [tickLabel](./circle/tickLabel)
RealReport-Chart 라이브러리 클래스 모델의 최상위 base 클래스.

  

### ranges: `IValueRange[]{:typescript}`
값 범위 목록.
범위별로 다른 스타일을 적용할 수 있다.  

### thickness: `string | number{:typescript}`
밴드의 두께를 픽셀 단위나, 
게이지 rim의 두께에 대한 상대적 크기로 지정할 수 있다.  

### visible: `boolean{:typescript}`
표시 여부.
모델에 따라 기본값이 다르다.
