## hand
게이지 바늘 설정 모델.
## Properties

### style
스타일셋 혹은 class selector.  

### length
바늘 길이.
픽셀 단위 크기나, 게이지 원호 반지름에 대한 상대 크기로 지정할 수 있다.  

### offset
바늘 중심과 게이지 중심 사이의 간격.
픽셀 단위 크기나, 게이지 원호 반지름에 대한 상대 크기로 지정할 수 있다.  

### radius
게이지 중심쪽의 바늘 반지름.
픽셀 단위 크기나, 게이지 원호 반지름에 대한 상대 크기로 지정할 수 있다.  

### visible: `boolean{:typescript}`
표시 여부.
모델에 따라 기본값이 다르다.
