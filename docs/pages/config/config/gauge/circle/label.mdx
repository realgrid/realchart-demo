## [label](/config/config/label)
게이지 중앙에 표시되는 label 설정 모델
## Properties

### animatable: `boolean{:js}`[#animatable]
게이지 값 변경 애니메이션이 실행될 때, label도 따라서 변경시킨다.  
<DefaultValue value="true" contents=""></DefaultValue>
### autoContrast: `boolean{:js}`[#autoContrast]
텍스트가 data point 내부에 표시되는 경우 포인트 색상과 대조되도록 표시한다.


밝게 표시할 때는 [lightStyle](#lightstyle) 을 적용하고,
어둡게 표시할 때는 [darkStyle](#darkstyle) 이 적용된다.
brightStyle이 지정되지 않으면 <Content type="expression" value="'rct-text-light'"></Content>이,
darkStyle이 지정되지 않으면 <Content type="expression" value="'rct-text-dark'"></Content>가 기본 적용된다.  
<DefaultValue value="true" contents=""></DefaultValue>

### numberFormat: `string{:js}`[#numberFormat]
label이 숫자일 때 표시 형식.  

### numberSymbols: `string{:js}`[#numberSymbols]
축의 tick 간격이 1000 이상인 큰 수를 표시할 때 
이 속성에 지정한 symbol을 이용해서 축약형으로 표시한다.  

### offsetX: `string | number{:js}`[#offsetX]
게이지 중심 등, label이 표시될 기준 위치에서 x 방향으로 이동한 픽셀 크기.
기준 위치는 게이지 종류에 따라 달라진다.  
<DefaultValue value="0" contents=""></DefaultValue>
### offsetY: `string | number{:js}`[#offsetY]
게이지 중심 등, label이 표시될 기준 위치에서 y 방향으로 이동한 픽셀 크기.
기준 위치는 게이지 종류에 따라 달라진다.  
<DefaultValue value="0" contents=""></DefaultValue>
