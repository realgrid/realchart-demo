## [series[type=barrange]](./series/barrange)
BarRange 시리즌.<br/>
수평 또는 수직 막대로 여러 값들의 범위들을 **비교**하는 데 사용한다.
막대의 길이가 값의 범위를 표시한다.<br/>
이 시리즈를 기준으로 생성되는 x축은 [category](/config/config/xAxis/category)이다.<br/><br/> [data](#data) 는 아래 형식들로 전달할 수 있다.<br/>
[주의] 데이터포인트 구성에 필요한 모든 값을 제공해야 한다.

###### 단일 값 및 값 배열
|형식|설명|
|---|---|
|y|단일 숫자면 low, y값. x 값은 순서에 따라 자동 결정.|
|[]|빈 배열이면 <Content type="keyword" value="null"></Content>. x 값은 데이터포인트 순서에 따라 자동 결정.|
|[z]|단일 값 배열이면 low, y값. x 값은 데이터포인트 순서에 따라 자동 결정.|
|[low, y]|두 값 배열이면 low값과 y값. x 값은 데이터포인트 순서에 따라 자동 결정.|
|[x, low, y,]|세 값 이상이면 순서대로 x, low, y값.<br/> 또는 [xField](#xfield) 속성이 숫자이면 x값의 index. [lowField](#lowfield) 는 low값의 index. [yField](#yfield) 는 y값의 index.|

###### json 배열
|Series 속성|설명|
|---|---|
| [xField](#xfield) |속성 값, 또는 <Content type="expression" value="'x', 'name', 'label'"></Content> 속성들 중 순서대로 값이 설정된 것이 x 값이 된다.|
| [yField](#yfield) |속성 값, 또는 <Content type="expression" value="'y', 'value'"></Content> 속성들 중 순서대로 값이 설정된 것이 y 값이 된다.|
| [lowField](#lowfield) |속성 값, 또는 <Content type="expression" value="'low'"></Content> 속성 값이 low 값이 된다.|
| [colorField](#colorfield) |속성 값, 또는 <Content type="expression" value="'color'"></Content> 속성 값으로 데이터포인트의 개별 색상으로 지정된다.|

## [series[type=scatter]](./series/scatter)
Scatter 시리즈.<br/>
데이터포인트를 shape로 표시되는 점 하나로 표시한다.<br/><br/> [data](#data) 는 아래 형식들로 전달할 수 있다.<br/>

###### 단일 값 및 값 배열
|형식|설명|
|---|---|
|y|단일 숫자면 y값. x 값은 순서에 따라 자동 결정.|
|[]|빈 배열이면 <Content type="keyword" value="null"></Content>. x 값은 순서에 따라 자동 결정.|
|[y]|값 하나인 배열이면 y값. x 값은 순서에 따라 자동 결정.|
|[x, y,]|두 값 이상이면 순서대로 x, y값.<br/> 또는 [xField](#xfield) 속성이 숫자이면 x값의 index. [yField](#yfield) 는 y값의 index.
 [colorField](#colorfield) 는 color값의 index.|

###### json 배열
|Series 속성|설명 |
|---|---|
| [xField](#xfield) |속성 값, 또는 <Content type="expression" value="'x', 'name', 'label'"></Content> 속성들 중 순서대로 값이 설정된 것이 x 값이 된다.|
| [yField](#yfield) |속성 값, 또는 <Content type="expression" value="'y', 'value'"></Content> 속성들 중 순서대로 값이 설정된 것이 y 값이 된다.|
| [colorField](#colorfield) |속성 값, 또는 <Content type="expression" value="'color'"></Content> 속성 값으로 데이터포인트의 개별 색상으로 지정된다.|
## [series[type=treemap]](./series/treemap)
treemap 시리즈.<br/>
차트나 split pane에 하나의 treemap만 존재할 수 있다.<br/><br/>

 [data](#data) 는 아래 형식들로 전달할 수 있다.<br/>
[주의] 데이터포인트 구성에 필요한 모든 값을 제공하지 않으면 <Content type="keyword" value="null"></Content>이 된다.
[주의] id, group 값은 문자열로 변환돼 사용된다.

###### 단일 값 및 값 배열
|형식|설명|
|---|---|
|y|단일 숫자면 low, y값. x 값은 순서에 따라 자동 결정.|
|[]|빈 배열이면 <Content type="keyword" value="null"></Content>. x 값은 순서에 따라 자동 결정.|
|[y, id, grup]|형식 설명 순서대로 값 결정. x 값은 데이터포인트 순서에 따라 자동 결정.|
|[x, y, id, group]|형깃 설명 순서대로 값 결정.<br/> 또는 [xField](#xfield) 속성이 숫자이면 x값, [yField](#yfield) 는 y값,<br/> [idField](#idfield) 는 id값, [groupField](#groupfield) 는 group값,<br/>.|

###### json 배열
|Series 속성|설명|
|---|---|
| [xField](#xfield) |속성 값, 또는 <Content type="expression" value="'x', 'name', 'label'"></Content> 속성들 중 순서대로 값이 설정된 것이 x 값이 된다.|
| [yField](#yfield) |속성 값, 또는 <Content type="expression" value="'y', 'value'"></Content> 속성들 중 순서대로 값이 설정된 것이 y 값이 된다.|
| [idField](#idfield) |속성 값, 또는 <Content type="expression" value="'id'"></Content> 속성 값이 length 값이 된다.|
| [groupField](#groupfield) |속성 값, 또는 <Content type="expression" value="'group'"></Content> 속성 값이 angle 값이 된다.|
