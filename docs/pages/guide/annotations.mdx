# 어노테이션(Annotations)

축과 시리즈만으로 대분분 데이터 표시 요구사항을 구현할 수 있지만,
불충분한 경우나 보다 직관적인 표현을 위해 annotation들을 사용할 수 있다.<br/>

## 설정

어노테이션은 chart와 body 두 군데에 설정할 수 있다.
차트 어노테이션들은 차트 설정의 최상위 '**annotations**'나 [annotation](/config/config/annotation) 항목으로 설정하고,
(*목록으로 설정하는 다른 구성 요소의 경우와 같이 단/복수 형 모두 가능하지만 복수형이 우선한다.*)
어노테이션의의 종류는 '**type**' 속성으로 지정한다.
특별히 지정하지 않으면 '**text**'로 자동 설정되지만, 명시적으로 종류를 지정하는 것이 좋다.
축이 둘 이상이면 배열로 지정하고, 하나면 배열 없이 지정해도 된다.<br/>
[body](/config/config/body)(시리즈들이 그려지는 영역)에 설정되는 어노테이션들은 body의 최상위 항목으로 설정한다. 

```js
const config = {
  annotations: [{
    type: 'image',
  }],
  body: {
    annotations: {
    }
  }
};
```

```js
const config = {
  annotation: {
  },
  body: {
    annotation: [{
        type: 'shape',
    }]
  }
};
```

현재 3가지 종류의 어노테이션이 있다.

##### Text 어노테이션

type은 '**text**'이고,
텍스트를 표시한다. 
기본 어노테이션으로서 설정에서 type을 지정하지 않으면 이 어노테이션으로 생성된다.<br/>
자세한 내용은 [가이드 설명](/guide/annotation-text)과 [설정](/config/config/annotation/text) 페이지를 참조한다.

##### Image 어노테이션

type은 '**image**'이고,
이미지를 표시한다. 
자세한 내용은 [가이드 설명](/guide/annotation-image)과 [설정](/config/config/annotation/image) 페이지를 참조한다.

##### Shape 어노테이션

type은 '**shape**'이고,
도형을 표시한다. 
자세한 내용은 [가이드 설명](/guide/annotation-shape)과 [설정](/config/config/annotation/shape) 페이지를 참조한다.
