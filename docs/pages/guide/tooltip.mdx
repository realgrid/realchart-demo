# 툴팁(Tooltip)과 Hovering

## Point Hovering

데이터포인트 **bar**나 **marker** 등의 view 위에 마우스가 올라갈(hover) 때, 
데이터포인트 view를 다른 것들과 구별되게 표시하여 사용자에게 해당 데이터포인트에 집중하게 하는 효과를 만들어낼 수 있다.
RealChart에서는 기본적으로 bar나 marker의 외곽선을 두터운 검정색으로 구분해서 표시한다.
