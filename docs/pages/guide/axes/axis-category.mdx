# Category Axis

축을 몇 개의 범주(category)로 나누고 범주에 연결된 데이터포인트들을 범주 내에 위치 시킨다.
기본적으로 각 범주의 너비는 동일하지만 개별적으로 다르게 표시할 수도 있다.
차트 설정에 x축의 종류를 지정하지 않으면 시리즈 종류에 따라 category 축으로 자동 설정될 수 있다.

## 설정

축의 종류를 "**category**"로 지정한다.

```
const config = {
    xAxis: {
        type: "category"
    }
};
```
