# 에셋(Assets)

시리즈나 게이지 또는 축 label 등을 효과적으로 표시할 때 참조하는 미리 정의된 스타일 요소들이다.

## Color List

차트 설정에서 에셋 type은 '**colors**'이다.

## Image List

에셋 type은 '**images**'이다.

## Linear Gradient

에셋 type은 '**lineargradient**'이다.

## Radial Gradient

에셋 type은 '**radialgradient**'이다.

## Pattern

에셋 type은 '**pattern**'이다.
