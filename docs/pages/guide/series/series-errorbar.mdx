# Error Bar 시리즈

주로 [bar](/guide/series/series-bar)시리즈와 함께 사용되며, 데이터의 변동성 및 불확실성을 표시하는 데 도움을 줄 수 있다.
오류(차) 값의 범위를 수직 막대로 표시한다.<br/>
시리즈 [type](/config/config/series#type)은 '**errorbar**'으로 지정한다.<br/>
이 시리즈를 기준으로 자동 생성되는 x축은 [카테고리](/config/config/xAxis/category)축이다.

## 설정

```js
const config = {
    series: [{
        data: [11, 22, 15, 9, 13, 27]
    }, {
        type: 'errorbar',
        pointLabel: true,
        data: [[10, 12], [20, 23], [14, 16], [8, 11], [12, 14], [25, 26]],
        color: '#333',
    }]
};
```
