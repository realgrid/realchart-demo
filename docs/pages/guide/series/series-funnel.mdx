# Funnel 시리즈

이어지는 단계를 나타내는 데 사용된다. 
단계들을 통과하는 과정에서 데이터가 어떻게 감소하거나 변화하는지를 직관적으로 보여준다.<br/> 
일반적으로 판매 과정, 마케팅 흐름, 이용자 행동 경로 등을 분석하는 데 사용될 수 있다.
맨 위에 가장 넓고(입력) 점차 아래쪽으로 좁아지는(결과) 모양을 가지고 있습니다. 
각 단계는 해당 단계를 완료한 데이터의 양을 나타내는 사각형 막대로 표시됩니다. 
이를 통해 각 단계에서의 이탈율이나 변환율을 시각적으로 확인할 수 있다.<br/>
시리즈 [type](/config/config/series#type)은 '**funnel**'로 지정한다.<br/>
Bar, Line 시리즈 등과 달리 축 설정이 필요 없는 시리즈이다.<br/>

<img src="/images/guide/series/funnel-01.png" width="70%" class="border-img"/>

## 설정

```js
const config = {
    series: {
        type: 'funnel',
        data: [
            { name: '웹사이트 방문', y: 13293 },   // [x(name), y]
            { name: '장바구니 담기', y: 4729 }, 
            { name: '결제제페이지', y: 2742}, 
            { name: '구매 완료', y: 1391},
        ]
    }
};
```
