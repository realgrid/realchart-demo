# Bubble 시리즈

[Scatter 시리즈](/guide/series/series-scatter)와 마찬가지로
**x**, **y**로 지정된 값들의 분포를 알려주는 것 외에, **z**로 지정된 추가 차원의 값을 거품 형태로 표시한다.
다양한 거품 크기와 색상 등의 스타일로 특정 값을 시각적으로 강조하는 데 사용할 수 있다.<br/>
시리즈 [type](/config/config/series#type)은 '**bubble**'로 지정한다.<br/>
이 시리즈를 기준으로 자동 생성되는 x축은 [선형](/config/config/xAxis/linear)축이다.
버블 차트에 category 축을 사용하는 것은 옳은 방법이 아니다.<br/>
