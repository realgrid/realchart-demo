# Area Range 시리즈

주로 시간이나 다른 축 변수에 의한 두 값 사이의 차이 변화를 보여주는 데 사용한다.<br/>
시리즈 [type](/config/config/series#type)은 '**arearange**'으로 지정한다.<br/>
이 시리즈를 기준으로 자동 생성되는 x축은 [선형](/config/config/xAxis/linear)축이다.<br/>

<img src="/images/guide/series/arearange-01.png" width="70%" class="border-img"/>

## 설정

```js
const config = {
    series: {
        type: 'arearange',
        data: [
                [20.6, 17.2], // [low, y(high)]
                [25.9, 18.7],
                [22.4, 17.7],
                [20.9, 14.1],
                [20.4, 15.1],
                [19.3, 6.6],
                [9.9, 3.6],
                [16.1, 1.8],
                ...
        ]
    }
};
```
