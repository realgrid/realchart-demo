# Waterfall 시리즈

[bar](/guide/series/series-bar)시리즈의 한 형태로서,
초기값에서 최종값으로 누적되는 변화를 각 단계의 수직 막대와 막대들을 연결하는 수평선으로 표시한다.<br/>
첫 막대는 초기값, 마지막 bar는 최종값을 나타내고,
단계별로 변화되는 값의 증감으로 별도의 색상으로 구분해서 표시한다.<br/>
시리즈 [type](/config/config/series#type)은 '**waterfall**'로 지정한다.<br/>
이 시리즈를 기준으로 자동 생성되는 x축은 [카테고리](/config/config/xAxis/category)축이다.
<img src="/images/guide/series/waterfall-01.png" width="70%" class="border-img"/>

## 설정

```js
const config = {
    series: {
        type: 'waterfall',
        data: [{
            name: 'Start',
            y: 120000
        }, {
            name: 'Product Revenue',
            y: 569000
        }, {
            name: 'Service Revenue',
            y: 231000
        }, {
            name: 'Positive Balance',
            isSum: true,
        }, {
            name: 'Fixed Costs',
            y: -342000
        }, {
            name: 'Variable Costs',
            y: -233000
        }, {
            name: 'Balance',
            isSum: true,
        }]
    }
};
```
