# Heatmap 시리즈

매트릭스 형태로 구성되는 데이터포인트들을 색상으로 구분한다. 주로 값이 클수록 짙은 색상으로 작을 수록 가벼운 색으로 표시한다.<br/>
시리즈 [type](/config/config/series#type)은 '**heatmap**'으로 지정한다.<br/>
이 시리즈를 기준으로 자동 생성되는 x축, y축 모두 [카테고리](/config/config/xAxis/category)축이다.

## 설정

```js
const config = {
    series: {
        type: 'heatmap',
        data: [
            [0, 0, -6.8],   // [x, y, z(color)]
            [0, 1, -5.4],
            [0, 2, -4.5],
            [0, 3, -3.9],
            [0, 4, -4],
            [0, 5, -7.6],
            [0, 6, -7],
            [1, 0, -3.6],
            [1, 1, -7],
            ...
        ]
    }
};
```
