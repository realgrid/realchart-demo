# Histogram 시리즈

데이터의 분포를 나타내는 데 사용되는 시리즈로서 연속된 데이터를 각각의 구간으로 나누고, 
각 구간에 속하는 데이터의 빈도수를 막대로 표시합니다. 
데이터의 분포를 쉽게 이해하고 비교할 수 있는 도구로서 통계학 등 다양한 분야헤서 널리 사용되는 강력한 도구이다.<br/>
시리즈 [type](/config/config/series#type)은 '**histogram**'으로 지정한다.<br/>
이 시리즈를 기준으로 자동 생성되는 x축은 [카테고리](/config/config/xAxis/category)축이다.
