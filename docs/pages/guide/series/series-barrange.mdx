# Bar Range 시리즈

축의 특점 지점 혹은 카테고리의 값의 범위를 표시하고, 
다른 지점 및 범주들과의 비교를 통해 데이터 분포나 변동성을 이해할 수 있도록 한다.<br/>
시리즈 [type](/config/config/series#type)은 '**barrange**'으로 지정한다.<br/>
이 시리즈를 기준으로 자동 생성되는 x축은 [카테고리](/config/config/xAxis/category)축이다.<br/>

<img src="/images/guide/series/barrange-01.png" width="70%" className="border-img"/>

## 설정

```js
const config = {
    series: {
        type: 'barrange',
        data: [
            [-13.9, 5.2], // [low, y(high)]
            [-16.7, 10.6],
            [-4.7, 11.6],
            [-4.4, 16.8],
            [-2.1, 27.2],
            [5.9, 29.4],
            [6.5, 29.1],
            [4.7, 25.4],
            [4.3, 21.6],
            [-3.5, 15.1],
            [-9.8, 12.5],
            [-11.5, 8.4]
        ]
    }
};
```
