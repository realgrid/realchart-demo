# Dumbbell 시리즈

[barrange](/guide/series/series-barrange) 시리즈의 한 형태로,
양 끝 값을 표시하는 점이나 원을 직선으로 연결하여 두 값 사이의 차이를 직관적으로 표시한다.
원의 끝이 아니라 중심이 값에 해당하는 위치다.<br/>
시리즈 [type](/config/config/series#type)은 '**dumbbell**'으로 지정한다.<br/>
이 시리즈를 기준으로 자동 생성되는 x축은 [카테고리](/config/config/xAxis/category)축이다.

## 설정

```js
const config = {
    series: {
        type: 'dumbbell',
        data: [
            [-13.9, 5.2],   // [low, y(high)]
            [-16.7, 10.6],
            [-4.7, 11.6],
            [-4.4, 16.8],
            [-2.1, 27.2],
            [5.9, 29.4],
            ...
        ]
    }
};
```
