# Lollipop 시리즈

[barr](/guide/series/series-bar) 시리즈의 한 형태로,
bar 대신 직선으로 표시하고 y값에 해당하는 위치에 원을 표시한다. 원의 끝이 아니라 중심이 값에 해당하는 위치다.<br/>
시리즈 [type](/config/config/series#type)은 '**lollipop**'으로 지정한다.<br/>
이 시리즈를 기준으로 자동 생성되는 x축은 [카테고리](/config/config/xAxis/category)축이다.

## 설정

```js
const config = {
    series: {
        type: 'lollipop',
        data: [155, 138, 122, 133, 114, 113],
    }
};
```
