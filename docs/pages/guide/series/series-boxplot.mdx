# Box Plot 시리즈

데이터 분포를 비교하고 이상 값을 발견하는데 사용되며,
5가지의 핵심 값을 한 눈에 확인할 수 있도록 한다.<br/>
시리즈 [type](/config/config/series#type)은 '**boxplot**'으로 지정한다.<br/>
이 시리즈를 기준으로 자동 생성되는 x축은 [카테고리](/config/config/xAxis/category)축이다.
