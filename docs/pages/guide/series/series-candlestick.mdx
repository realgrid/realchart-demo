# Candlestick 시리즈

주식 시장 등에서 가격의 변화를 보여주는 데 주로 사용한다.
각 캔들은 해당 기간 동안의 가격 움직임을 시가, 종가, 최저가, 최고가를 표시해서 보여준다.<br/>
시리즈 [type](/config/config/series#type)은 '**candlestick**'으로 지정한다.<br/>
이 시리즈를 기준으로 자동 생성되는 x축은 [카테고리](/config/config/xAxis/category)축이다.

<img src="/images/guide/series/candlestick-01.png" width="70%" className="border-img"/>

## 설정

```js
const config = {
    series: {
        type: 'candlestick',
        data: [
            [560, 651, 748, 895], // [low, open, high, y(close)]
            [533, 753, 939, 980],
            [514, 662, 817, 870],
            [624, 802, 816, 871],
            [634, 736, 804, 882],
            ...
        ]
    }
};
```
