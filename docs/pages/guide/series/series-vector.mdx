# Vector 시리즈

x, y로 지정된 데이터포인트에 길이와 방향(각도)를 지정해서 화살표로 표시한다.
힘, 속도, 방향 등과 같이 벡터로 표현되는 데이터를 시각화하는 데 사용될 수 있다.<br/>
시리즈 [type](/config/config/series#type)은 '**vetor**'로 지정한다.<br/>
이 시리즈를 기준으로 자동 생성되는 x축은 [선형](/config/config/xAxis/linear)축이다.
