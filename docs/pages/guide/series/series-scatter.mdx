# Scatter 시리즈

**x**, **y**로 지정된 두 값 사이의 관계와 분포를 알려준다.<br/>
시리즈 [type](/config/config/series#type)은 '**scatter**'로 지정한다.<br/>
이 시리즈를 기준으로 자동 생성되는 x축은 [선형](/config/config/xAxis/linear)축이다.
