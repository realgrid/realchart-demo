# Treemap 시리즈

트리 구조의 데이터를 사각형으로 표시되는 node로 표시한다.
각 노드는 자식 노드들을 포함할 수 있으면 부모 노드의 영역 내에 배분되어 표시된다.
주로 계층적 데이터의 크기, 구조, 상대적인 비율을 이해하고 비교하는 데 사용된다.<br/>
시리즈 [type](/config/config/series#type)은 '**treemap**'으로 지정한다.<br/>
[pie](/guide/series/series-pie), [funnel](/guide/series/series-funnel) 시리즈와 마찬가지로 축 설정이 필요하지 않다.
