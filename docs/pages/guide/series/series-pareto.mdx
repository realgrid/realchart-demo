# Pareto 시리즈

[line](/guide/sieries/series-line)시리즈의 한 형태로,
주로 데이터 기준이 되는 [bar](/guide/series/series-pareto)시리즈와 함꼐 표시된다.
데이터 그룹 상대적으로 어떤 것들이 가장 중요한 지를 직관적으로 얄려주는 데 사용된다.<br/>
시리즈 [type](/config/config/series#type)은 '**pareto**'로 지정한다.<br/>
이 시리즈를 기준으로 자동 생성되는 x축은 [카테고리](/config/config/xAxis/category)축이다.

## 설정

```js
const config = {
    series: {
        type: 'pareto',
        data: [155, 138, 122, 133, 114, 113],
    }
};
```
