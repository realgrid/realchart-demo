# Image Annotation

[imageUrl](/config/config/annotation/image#imageurl)로 지정한 이미지를 
차트나 [body](/config/config/body)(시리즈가 표시되는 plotting 영역)의 지정한 위치에 표시한다.

<img src="/images/guide/annotation-image-01.png" width="80%" class="border-img"/>

[imageUrl](/config/config/annotation/image#imageurl) 속성에 표시할 이미지 경로를 설정한다.<br/>
왼쪽 위 첫쩨 이미지에는 [backgroundStyle](/config/config/annotation/image#backgroundstyle)을 설정했다.<br/>
오른쪽 위 두번째 이미지는 [front](/config/config/annotation/image#front)가 true로 설정돼서 시리즈 위에 표시되고 있다.<br/>
위 오른쪽에서 세번째 큰 이미지는 [noClip](/config/config/annotation/image#noclip)이 true로 설정돼서 body 영역 바깥까지 표시되고 있다.<br/>
왼쪽 아래는 [noClip](/config/config/annotation/image#noclip)이 설정되지 않아 body 영역에 클립되고 있다.<br/>

## 설정

'**type**'을 'image'로 지정한다.

```js
    anntations: [{
        type: 'image',
        imageUrl: '/images/chart.png',
        rotation: -20
    }]
```

'**type**'을 지정하지 않고, 
[imageUrl](/config/config/annotation/image#imageurl)을 지정하면
'image'로 지정한 것과 동일하다.

```js
    anntations: [{
        imageUrl: '/images/chart.png',
    }]
```
