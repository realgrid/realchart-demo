# Shape Annotation

[shape](/config/config/annotation/text#shape) 속성으로 지정한 도형을 
차트나 [body](/config/config/body)(시리즈가 표시되는 plotting 영역)의 지정한 위치에 표시한다.

<img src="/images/guide/annotation-shape-01.png" width="80%" class="border-img"/>

## 설정

'**type**'을 'image'로 지정한다.

```js
    anntations: [{
        type: 'image',
        shape: 'rectangle',
        rotation: -20,
        style: {
            stroke: 'red',
            strokeWidth: '3px'
        }
    }]
```
