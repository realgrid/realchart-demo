# RealChart 개요

RealChart는 웹 어플리케이션 개발을 위한 차트 컴포넌트 라이브러리이다.

![차트](/images/docs/realchart.png)

기본적으로 2차원 평면 위에 연관된 데이터를 다양한 방식으로 표시하며,
관련된 정보들을 알려주는 여러 구성 요소들을 포함한다.

![차트 구성 3](/images/docs/realchart-parts-03.png)
![차트 구성 1](/images/docs/realchart-parts-01.png)
![차트 구성 2](/images/docs/realchart-parts-02.png)
