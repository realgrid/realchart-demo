# RealChart 개요

RealChart는 웹 어플리케이션 개발을 위한 차트 컴포넌트 라이브러리이다.

![차트](/images/docs/realchart.png)

기본적으로 2차원 평면 위에 연관된 데이터를 다양한 방식으로 표시하며,
관련된 정보들을 알려주는 여러 구성 요소들을 포함한다.

![차트 구성 요소들](/images/docs/realchart-parts.jpg)

## Axis

리얼차트는 2차원 [직교 좌표 평면](https://ko.wikipedia.org/wiki/%EB%8D%B0%EC%B9%B4%EB%A5%B4%ED%8A%B8_%EC%A2%8C%ED%91%9C%EA%B3%84) 위에 데이터 포인트들을 표시한다.
차트에서는 기본적으로 좌측 아래에 원점이 있으며 아래쪽에 수평으로 X축, 왼쪽 수직으로 Y축을 명시적으로 표시한다.
X축은 원점에서 오른쪽으로, Y축은 원점에서 위쪽으로 값이 증가한다.\
때때로 원점이나 축 방향을 바꿔서 표시하는 것이 효과적일 수도 있는데,
RealChart는 [inverted](/config/config#inverted), [reversed](/config/config/yAxis/linear#reversed) 속성 등을 사용하여 좌표계의 구성을 다르게 설정할 수 있다.\
또, [polar](/config/config#polar)를 true로 지정해서 직교 대신 [극좌표계](https://ko.wikipedia.org/wiki/%EA%B7%B9%EC%A2%8C%ED%91%9C%EA%B3%84) 위에 데이터 포인트들을 표시할 수 있다.

![Polar 차트](/images/docs/polar.png)
(현재, polar 차트로 표시할 수 있는 시리즈 종류는 
['bar'](/config/config/series/bar), 
['line'](/config/config/series/line), 
['area'](/config/config/series/area), 
['bubble'](/config/config/series/bubble), 
['scatter'](/config/config/series/scatter) 
이다.)

차트에서는 축 선과, 텍스트와 선분으로 구성되는 몇 개의 틱(tick)을 표시해서 데이터 포인트의 위치를 직관적으로 구분할 수 있게 한다.
축 별로 타이틀을 별도로 표시할 수도 있다.

축은 값이 분포되는 방식에 따라 몇가지 종류로 구분된다.

### Linear Axis

균일한 간격으로 값들이 배치된다.

### Log Axis

로그값 배율로 값들이 배치된다.

### Time Axis

1밀리초를 1로 환산해서 균일한 간격으로 배치된다.

### Category Axis

지정한 개수만큼 축을 분할한다.

## Series

관련된 하나 이상의 값들을 같은 x/y축과 공통 속성을 적용해서 표시한다.
개별 값들을 **데이터포인트**라고 하고, 각 데이터포인트들을 표시하는 방식에 따라 여러 종류의 시리즈가 존재한다.

## Title

차트 제목 및 부제목을 표시한다.

## Legend

현재 차트에 표시된 시리즈 혹은 데이터포인트들을 범례 항목들로 표시하고,
범례 항목들을 클릭해서 연결된 시리즈나 데이터포인트를 감추거나 다시 표시되게 할 수 있다.

## Annotation

데이터포인트 시리즈 외에 부가적인 정보를 표시할 필요가 있을 때,
사용할 수 있는 차트 아이템이다. 텍스트, 이미지, Shape 등을 적절하게 표시할 수 있다.
