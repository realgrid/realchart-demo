# Clock 게이지

현재 시각이나 지정한 시간을 표시하는 아날로그 시계 게이지.<br/>
기본적으로 현재 시간을 표시하지만, [시간](/config/config/gauge/clock#time)을 직접 지정할 수 있다.
[시간대(timezone)](/config/config/gauge/clock#timezon)을 설정할 수도 있다.<br/>
아래 그림처럼 시계 [테두리(rim)](/config/config/gauge/clock/rim)나 
[침(hand)](/config/config/gauge/clock/hourhand), 
[틱(tick)](/config/config/gauge/clock/tick) 등을 개별적으로 스타일링 할 수 있다.

<img src="/images/guide/gauge-clock-01.png" width="80%" class="border-img"/>

## 설정

type은 '**clock**'으로 지정하고, 
현재 시각 대신 특정한 시각을 표시하고 싶은 경우 [time](/config/config/gauge/clock#time) 속성에 
[밀리초 기준 숫자](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Date/getTime)나, 
[Date](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Date)형 값, 
혹은 javascript가 해석할 수 있는 문자열로 시간을 지정한다.

```js
{
    type: 'clock',
    time: '2024-01-11T11:12:13', // 지정하지 않으면 현재 시간
    tick: {},
    minorTick: {},      // 시간 사이에 표시되는 작은 tick
    rim: {},            // 테두리
    hourHand: {},       // 시침
    minuteHand: {},     // 분침
    secondeHand: {},    // 초침
    pin: {},            // 중심 핀
    tickLabel: {},
    label: {},
    style: {}
}
```
