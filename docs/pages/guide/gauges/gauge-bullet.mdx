# Bullet 게이지

Bullet 게이지.<br/>
현재 값을 목표 값과 비교해서 표시한다.
현재 값의 위치는 [최소값](/config/config/gauge/bullet#minvalue)과 [최대값](/config/config/gauge/bullet#maxvalue)에 따라 정해진다. 
기본값은 각각 0, 100이다.<br/>

<img src="/images/guide/gauge/bullet-01.png" width="70%" class="border-img"/>

## 설정

type은 '**bullet**'로 지정하고, 
반드시 [현재값(value)](/config/config/gauge/bullet#value)과
[목표값(targetValue)](/config/config/gauge/bullet#targetValue) 속성들을 지정해야 한다.

```js
{
    type: 'bullet',
    value: 121,
    targetValue: 150,
    maxValue: 200,
    band: {},
    scale: {},
    targetBar: {},
    valueBar: {},
    ranges: [],
    label: {},
    style: {}
}
```
