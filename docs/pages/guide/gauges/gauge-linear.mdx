# Linear 게이지

선형 게이지.<br/>
현재 값의 위치는 [최소값](/config/config/gauge/bullet#minvalue)과 [최대값](/config/config/gauge/bullet#maxvalue)에 따라 정해진다. 
기본값은 각각 0, 100이다.<br/>

## 설정

type은 '**linear**'로 지정하고, 반드시 현재 값을 [value](/config/config/gauge/linear#value) 속성에 지정해야 한다.

```js
{
    type: 'linear',
    value: 121,
    maxValue: 200
}
```
