## 차트 테스트

import { RealChartReact } from "../../components/RealChart/RealChartReact";
import { config } from "../../template/chart1";

- 차트를 올리고 Config를 지정할 수 있습니다.
- Config 수정 후 <Button compact variant="outline">적용</Button> 버튼을 누르면 변경된 코드를 차트에 적용합니다.

<RealChartReact configString={"const config = " + JSON.stringify(config)} showEditor={true} autoUpdate={false} />
<Button>Button</Button>
