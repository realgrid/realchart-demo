# 분할 모드(Split)

RealChart는 기본적으로 [body](/config/config/body)라 불리우는 하나의 plotting 영역에 
차트에 설정된 모든 시리즈들을 표시한다. 마찬가지로 x, y축 모두 이 영역을 중심으로 배치된다.

<img src="/images/guide/body-01.png" width="50%" className="border-img"/>

Split Mode는 차트를 수평 및 수직으로 분할해서 각 분할마다 별도의 body 영역이 생성되도록 한다.
또 body 영역별로 자신의 x, y축이 표시될 수 있다.

<img src="/images/guide/body-split.png" width="50%" className="border-img"/>

차트 최상위 [split](/config/config/split) 모델로 설정하며,
먼저 [visible](/config/config/split#visible)을 **true**로 지정하고, 
[rows](/config/config/split#rows), [cols](/config/config/split#cols) 속성에 
각각 y축, x축 방향 분할 수를 지정한다.

```js
const config = {
    split: {
        visible: true,  // 반드시 true로 지정해야 한다.
        rows: 2,
        cols: 3
    }
};
```

## 축

각 분할은 영역 주위에 별도의 x, y축들을 표시할 수 있다.
축들이 어느 분할 주위에 표시될 지는 Axis.[row](/config/config/base/axis#row), 
[col](/config/config/base/axis#col) 속성으로 설정한다.

```js
const config = {
    split: {
        visible: true,
        cols: 3
    },
    xAxis: [{
        // col을 지정하지 않으면 0,
        // row를 지정하지 않으면 0
    }, {
        col: 1,
    }, {
        col: 2,
    }],
}
```

## 시리즈

분할 모드에서 시리즈가 어느 body에 표시될 지는 Series.[row](/config/config/base/series#row), 
Series.[col](/config/config/base/series#col) 속성으로 직접 지정하거나,
시리즈에 연결된 x, y축에 의해 자동으로 위치가 결정된다.

```js
const config = {
    series: [{
        // xAxis나 col을 모두 지정하지 않으면 col = 0
        // yAxis나 row를 모두 지정하지 않으면 row = 0
    }, {
        xAxis: 1,
    }, {
        col: 2,
    }]
}
```
