# 축(Axis)

## 2차원 평면

RealChart를 포함한 차팅 툴은 기본적으로 2차원 평면 위에 데이터를 표시한다.
2차원 평면은 X/Y 두 축으로 구성되며, 
두 축이 직각으로 교차하는 직교 좌표, X축의 각도를 기준으로 표시 지점을 결정하는 극좌표가 있다.

## 설정

차트 설정에서 두 축은 각각 'xAxis', 'yAxis' 이름으로 포함된다.
특별히 지정하지 않으면 y축은 'linear', x축은 시리즈 종류에 따라 자동 설정된다.
