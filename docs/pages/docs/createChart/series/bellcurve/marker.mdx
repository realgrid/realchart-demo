## marker

## Properties

### firstVisible
첫번째 point의 marker 표시 여부.  
`default: MarkerVisibility.DEFAULT`  
- `default` visible 속성에 따른다.
- `hidden` visible 속성과 상관없이 항상 표시하지 않는다.
- `visible` visible 속성과 상관없이 항상 표시한다.
### lastVisible
첫번째 point의 marker 표시 여부.  
`default: MarkerVisibility.DEFAULT`  
- `default` visible 속성에 따른다.
- `hidden` visible 속성과 상관없이 항상 표시하지 않는다.
- `visible` visible 속성과 상관없이 항상 표시한다.
### maxVisible
최대값 point들의 marker 표시 여부.  
`default: MarkerVisibility.DEFAULT`  
- `default` visible 속성에 따른다.
- `hidden` visible 속성과 상관없이 항상 표시하지 않는다.
- `visible` visible 속성과 상관없이 항상 표시한다.
### minVisible
최소값 point들의 marker 표시 여부.  
`default: MarkerVisibility.DEFAULT`  
- `default` visible 속성에 따른다.
- `hidden` visible 속성과 상관없이 항상 표시하지 않는다.
- `visible` visible 속성과 상관없이 항상 표시한다.
### radius: number
`default: 4`  

### shape
명시적으로 지정하지 않으면 typeIndex에 따라 Shapes 중 하나로 돌아가면서 설정된다.  
- `circle` 
- `diamond` 
- `itriangle` 
- `rectangle` 
- `square` 
- `star` 
- `triangle`
