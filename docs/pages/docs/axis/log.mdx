## axis.log
이 축에 연결된 시리즈들의 point y값을 log10으로 계산된 위치에 표시한다.
y축으로만 사용될 수 있다.
## Properties

### max: number
명시적으로 지정하는 최대값.
축에 연결된 data point들의 값으로 계산된 최대값 대신 이 값이 축의 최소값이 된다.
 maxPadding 도 무시된다.

### min: number
명시적으로 지정하는 최소값.
축에 연결된 data point들의 값으로 계산된 최소값 대신 이 값이 축의 최소값이 된다.
 minPadding 도 무시된다.

### nullable: boolean
data point의 이 축 값이 NaN일 때도 point를 표시할 지 여부.

### position: AxisPosition
표시 위치.

### reversed: boolean
true면 반대 방향으로 point 위치들이 지정된다.

### tickBase: string | number
tick 개수를 맞춰야 하는 대상 axis.
base의 strictMin, strictMax가 설정되지 않아야 한다.
base의 startFit, endFilt의  AxisFit.TICK 으로 설정되어야 한다.
