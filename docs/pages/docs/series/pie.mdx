## series.pie

## Properties

### autoSlice: boolean
클릭한 데이터 포인트를 slice 시킨다.
기존 slice 됐던 포인트는 원복된다.  
`default: true` 
### borderRadius: number
`default: 0` 
### center: RtPercentSize

### centerX: RtPercentSize

### centerY: RtPercentSize

### clipped: boolean
body 영역을 벗어난 data point view는 잘라낸다.  
`default: false` 
### color: string
데이터 포인트 기본 색.  

### colorField: string
undefined이면, data point의 값이 객체일 때 'color'.  

### dataProp: string
undefined이면 "data".  

### displayInLegend: boolean
명시적 false로 지정하면 legend에 표시하지 않는다.  
`default: true` 
### groupSize: number
`default: 1` 
### innerSize: RtPercentSize
0보다 큰 값을 지정해서 도넛 형태로 표시할 수 있다.
 innerText 로 도넛 내부에 표시될 텍스트를 지정할 수 있다.  

### innerText: PieSeriesText
innerSize 가 0보다 클 때, 도넛 내부에 표시되는 텍스트.
기본 클래스 selector는 <b>'rct-pie-series-inner'</b>이다.  
`default: ...` 
### labelDistance: number
`default: 25` 
### legendByPoint: boolean
데이터 포인트별 legend 항목을 표시한다.  
`default: false` 
### pointColors: boolean | string[]
데이터 포인트별 색들을 지정한다.
false로 지정하면 모든 포인트들이 시리즈 색으로 표시된다.
true로 지정하면 기본 색들로 표시된다.
색 문자열 배열로 지정하면 포함된 색 순서대로 표시된다.
undefined나 null이면 시리즈 종류에 따라 false 혹은 true로 해석된다.  

### size: RtPercentSize
시리즈 본체의 크기.


픽셀 크기나 차지할 수 있는 전체 크기에 대한 상대적 크기로 지정할 수 있다.  
`default: RadialSeries.SIZE` 
### sliceDuration: number
Slice animation duration.
밀리세컨드(ms) 단위로 지정.  
`default: 300` 
### sliceOffset: RtPercentSize
`default: '7%'` 
### startAngle: number
`default: 0` 
### xAxis: string | number

### xField: string | number
undefined이면, data point의 값이 array일 때는 0, 객체이면 'x'.  

### xStart: string | number
x축 값이 설정되지 않은 첫번째 데이터 point에 설정되는 x값.
이 후에는  xStep 씩 증가시키면서 설정한다.
'time' 축일 때, 정수 값 대신 시간 단위('day', 'week', 'month', 'year')로 지정할 수 있다.
숫자로 지정하면 1은 1밀리초로 지정된다. 
이 속성이 지징되지 않은 경우  ChartOptions.xStart 가 적용된다.  

### xStep: string | number
x축 값이 설정되지 않은 데이터 point에 지정되는 x값의 간격.
첫번째 값은  xStart 로 설정한다.
time 축일 때, 정수 값 대신 시간 단위('y', 'm', 'd', 'h', 'n', 's')로 지정할 수 있다.
이 속성이 지정되지 않으면  ChartOptions.xStep 이 적용된다.  

### yAxis: string | number

### yField: string | number
undefined이면, data point의 값이 array일 때는 1, 객체이면 'y'.  

### zOrder: number
`default: 0`
