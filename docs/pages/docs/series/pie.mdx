## series.pie

## Properties

### borderRadius: number

### centerX: number

### centerY: number

### clipped: boolean
body 영역을 벗어난 data point view는 잘라낸다.

### color: string
데이터 포인트 기본 색.

### colorField: string
undefined이면, data point의 값이 객체일 때 'color'.

### dataProp: string
undefined이면 "data".

### exclusive: boolean
true이면 섹터 하나만 마우스 클릭으로 sliced 상태가 될 수 있다.
Point의 sliced 속성을 직접 지정하는 경우에는 이 속성이 무시된다.

### groupSize: number

### innerSize: RtPercentSize
0보다 큰 값을 지정해서 도넛 형태로 표시할 수 있다.

### labelDistance: number

### pointColors: boolean | string[]
데이터 포인트별 색들을 지정한다.
false로 지정하면 모든 포인트들이 시리즈 색으로 표시된다.
true로 지정하면 기본 색들로 표시된다.
색 문자열 배열로 지정하면 포함된 색 순서대로 표시된다.
undefined나 null이면 시리즈 종류에 따라 false 혹은 true로 해석된다.
- [기본 샘플](https://jsfiddle.net/gh/get/library/pure/realgrid/realchart-demo/tree/master/demo/bar/bar)
- [다른 샘플](https://jsfiddle.net/gh/get/library/pure/realgrid/realchart-demo/tree/master/demo/bar/bar-null)

### size: RtPercentSize
원형 플롯 영역의 크기.


픽셀 크기나 차지할 수 있는 전체 크기에 대한 상대적 크기로 지정할 수 있다.

### sliceDuration: number
Slice animation duration.
밀리세컨드 단위로 지정.

### sliceOffset: RtPercentSize

### startAngle: number

### xAxis: string | number

### xField: string | number
undefined이면, data point의 값이 array일 때는 0, 객체이면 'x'.

### xStart: string | number
x축 값이 설정되지 않은 첫번째 데이터 point에 설정되는 x값.
이 후에는  xStep 씩 증가시키면서 설정한다.
'time' 축일 때, 정수 값 대신 시간 단위('day', 'week', 'month', 'year')로 지정할 수 있다.
숫자로 지정하면 1은 1밀리초로 지정된다. 
이 속성이 지징되지 않은 경우  ChartOptions.xStart 가 적용된다.

### xStep: string | number
x축 값이 설정되지 않은 데이터 point에 지정되는 x값의 간격.
첫번째 값은  xStart 로 설정한다.
time 축일 때, 정수 값 대신 시간 단위('day', 'week', 'month', 'year')로 지정할 수 있다.
이 속성이 지정되지 않으면  ChartOptions.xStep 이 적용된다.

### yAxis: string | number

### yField: string | number
undefined이면, data point의 값이 array일 때는 1, 객체이면 'y'.

### zOrder: number
