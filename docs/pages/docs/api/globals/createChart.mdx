### createChart(doc, container, config, animate, callback?): [RcChartControl](../classes/RcChartControl)[#createChart]
차트 설정 모델을 기빈으로
[Chart 컨트롤](/docs/api/classes/RcChartControl) svg를 생성한 후,
지정된 div 엘리먼트의 유일한 자식으로 포함시킨다.\
div 전체 영역의 크기로 표시된다.

```js
const chart = RealChart.createChart(document, 'realchart', config);
```



| Parameter | Type | Description |
| ----------- | ----------- | ----------- |
| doc | Document | N/A |
| container | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) &#124; HTMLDivElement | 컨트롤이 생성되는 div 엘리먼트나 id |
| config | [any](https://www.typescriptlang.org/docs/handbook/2/everyday-types.html#any) | 차트 모델 설정 JSON |
| animate | [boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean) | 첫 로딩 animation을 실행한다. |
| callback | ( &#124; ...args &#124; [unknown](https://www.typescriptlang.org/docs/handbook/release-notes/typescript-3-0.html#new-unknown-top-type) &#124; \[ &#124; ] &#124; ) &#124;   &#124; => &#124;   &#124; [void](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/undefined) | N/A |
