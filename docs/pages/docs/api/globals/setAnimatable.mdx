### setAnimatable(value): [void](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/undefined)
false로 지정하면 시리즈 시작 애니메이션 등을 포함,
모든 애니메이션을 실행하지 않는다.\
기본은 animation이 가능한 상태다.

```
RealChart.setAnimatable(false);
```



| Parameter | Type | Description |
| ----------- | ----------- | ----------- |
| value | [boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean) | 애니메이션 실행 여부 |
