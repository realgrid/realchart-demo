### setDebugging(debug): [void](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/undefined)
true로 지정하면 차트 요소별 디버그 경계를 표시한다.\
기본은 false 상태다.

```js
RealChart.setDebugging(true);
```



| Parameter | Type | Description |
| ----------- | ----------- | ----------- |
| debug | [boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean) | 디버깅 상태 표시 여부 |
