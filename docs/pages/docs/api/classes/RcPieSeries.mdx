## RcPieSeries
            ### Extends
- [RcChartSeries](../classes/RcChartSeries)
            
            
**'pie'** 시리즈.

            



## Config Properties
### autoSlice: `boolean{:js}`[#autoSlice]
클릭한 데이터 포인트를 slice 시킨다.
기존 slice 됐던 포인트는 원복된다.

### borderRadius: `number{:js}`[#borderRadius]


### center: `string | number{:js}`[#center]


### centerX: `string | number{:js}`[#centerX]


### centerY: `string | number{:js}`[#centerY]


### clipped: `boolean{:js}`[#clipped]
body 영역을 벗어난 data point view는 잘라낸다.

### clockwise: `boolean{:js}`[#clockwise]
true면 시계 방향으로 회전한다.

### color: `string{:js}`[#color]
데이터 포인트 기본 색.

### colorField: `string{:js}`[#colorField]
undefined이면, data point의 값이 객체일 때 'color'.

### dataProp: `string{:js}`[#dataProp]
undefined이면 "data".

### groupSize: `number{:js}`[#groupSize]


### innerRadius: `string | number{:js}`[#innerRadius]
0보다 큰 값을 지정해서 도넛 형태로 표시할 수 있다.
시리즈 원호의 반지름에 대한 상대적 크기나 픽셀 수로 지정할 수 있다.
 innerText 로 도넛 내부에 표시될 텍스트를 지정할 수 있다.

### innerText: `PieSeriesText{:js}`[#innerText]
innerRadius 가 0보다 클 때, 도넛 내부에 표시되는 텍스트.
기본 클래스 selector는 <b>'rct-pie-series-inner'</b>이다.

### labelDistance: `number{:js}`[#labelDistance]


### legendByPoint: `boolean{:js}`[#legendByPoint]
데이터 포인트별 legend 항목을 표시한다.

### onPointClick: `PointClickCallbck{:js}`[#onPointClick]
데이터 point가 클릭될 때 발생하는 이벤트 콜백.
명시적 true를 리턴하면 기본 동작이 진행되지 않는다.

### pointColors: `boolean | string[]{:js}`[#pointColors]
데이터 포인트별 색들을 지정한다.
false로 지정하면 모든 포인트들이 시리즈 색으로 표시된다.
true로 지정하면 기본 색들로 표시된다.
색 문자열 배열로 지정하면 포함된 색 순서대로 표시된다.
undefined나 null이면 시리즈 종류에 따라 false 혹은 true로 해석된다.

### pointStyleCallback: `PointStyleCallback{:js}`[#pointStyleCallback]
데이터 point의 동적 스타일 콜백.

### radius: `string | number{:js}`[#radius]
시리즈 원호의 반지름.
픽셀 크기나 plot 영역 크기에 대한 상대적 크기로 지정할 수 있다.
'50%'로 지정하면 plot 영역의 width나 height중 작은 크기와 동일한 반지름으로 표시된다.

### ranges: `IValueRange[]{:js}`[#ranges]
값 범위 목록.
범위별로 다른 스타일을 적용할 수 있다.

### sliceDuration: `number{:js}`[#sliceDuration]
Slice animation duration.
밀리세컨드(ms) 단위로 지정.

### sliceOffset: `string | number{:js}`[#sliceOffset]


### startAngle: `number{:js}`[#startAngle]
시리즈 원호 시작 각도.
지정하지 않거나 잘못된 값이면 0으로 계산된다.
0은 시계의 12시 위치다.

### style: `SVGStyles | string{:js}`[#style]
스타일셋 혹은 class selector.

### totalAngle: `number{:js}`[#totalAngle]
시리즈 원호 전체 각도.
0 ~ 360 사이의 값으로 지정해야 한다.
범위를 벗어난 값은 범위 안으로 조정된다.
지정하지 않거나 잘못된 값이면 360으로 계산된다.

### visibleInLegend: `boolean{:js}`[#visibleInLegend]
명시적 false로 지정하면 legend에 표시하지 않는다.

### xAxis: `string | number{:js}`[#xAxis]


### xField: `string | number{:js}`[#xField]
undefined이면, data point의 값이 array일 때는 0, 객체이면 'x'.

### xStart: `any{:js}`[#xStart]
x축 값이 설정되지 않은 첫번째 데이터 point에 설정되는 x값.
이 후에는  xStep 씩 증가시키면서 설정한다.
이 속성이 지징되지 않은 경우  ChartOptions.xStart 가 적용된다.

### xStep: `string | number{:js}`[#xStep]
x축 값이 설정되지 않은 데이터 point에 지정되는 x값의 간격.
첫번째 값은  xStart 로 설정한다.
time 축일 때, 정수 값 대신 시간 단위('y', 'm', 'd', 'h', 'n', 's')로 지정할 수 있다.
이 속성이 지정되지 않으면  ChartOptions.xStep 이 적용된다.

### yAxis: `string | number{:js}`[#yAxis]


### yField: `string | number{:js}`[#yField]
undefined이면, data point의 값이 array일 때는 1, 객체이면 'y'.

### zOrder: `number{:js}`[#zOrder]


### visible: `boolean{:js}`[#visible]
표시 여부.

## Methods
### get(prop): [any](https://www.typescriptlang.org/docs/handbook/2/everyday-types.html#any)[#get]
지정한 설정 값(들)을 가져온다.\
지정 가능한 설정 값 목록은 [Configuration API 페이지](/config)에서 확인할 수 있다. 

```js
console.log(chart.series.get('name'))
```

하위 객체의 설정 속성은 속성 경로를 지정해서 직접 가져올 수도 있다.

```js
console.log(chart.axis.get('title.text'));
```



| Parameter | Type | Description |
| ----------- | ----------- | ----------- |
| prop | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | 속성 경로. |
### set(prop, value, redraw): [RcChartObject](../classes/RcChartObject)[#set]
지정한 속성의 값(들)을 설정한다.\
지정 가능한 설정 값 목록은 [Configuration API 페이지](/config)에서 확인할 수 있다.\
또, 이 객체 자신을 리턴하므로 javascript에서 builder 패턴으로 설정 값들을 연속해서 지정할 수 있다.

```js
 chart.series.set('xAxis', 1).set('yAxis', 1);
```

[get](#get)에서와 마찬가지로 하위 객체의 설정 속성을 속성 경로를 통해 직접 설정할 수도 있다.

```js
chart.axis.set('title.text', 'New Title'));
```



| Parameter | Type | Description |
| ----------- | ----------- | ----------- |
| prop | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | 설정 속성 경로 문자열 |
| value | [any](https://www.typescriptlang.org/docs/handbook/2/everyday-types.html#any) | 지정할 값(들) |
| redraw=`true` | [boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean) | true로 지정하면 chart를 다시 그린다.                false로 지정하고 여러 설정 후에 [render](../classes/RcChartControl#render)를 호출해서 다시 그리게 할 수도 있다. |
### setAll(props, redraw): [RcChartObject](../classes/RcChartObject)[#setAll]
JSON 객체로 지정한 속성들의 값을 설정한다.\
지정 가능한 설정 값 목록은 [Configuration API 페이지](/config)에서 확인할 수 있다. 

```js
 chart.series.set({
     xAxis: 1,
     yAxis: 1,
     yField: 'salary'
 });
```



| Parameter | Type | Description |
| ----------- | ----------- | ----------- |
| props | [object](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object) | 여러 설정 값들이 지정된 JSON 객체 |
| redraw=`true` | [boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean) | true로 지정하면 chart를 다시 그린다.                false로 지정하고 여러 설정 후에 [render](../classes/RcChartControl#render)를 호출해서 다시 그리게 할 수도 있다. |
### toggle(prop, redraw): [RcChartObject](../classes/RcChartObject)[#toggle]
Boolean 타입의 설정값을 변경한다.\
지정 가능한 설정 값 목록은 [Configuration API 페이지](/config)에서 확인할 수 있다. 

```js
chart.getSeries('ser02').toggle('visible');
```



| Parameter | Type | Description |
| ----------- | ----------- | ----------- |
| prop | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | 설정 속성 이름 |
| redraw=`true` | [boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean) | true로 지정하면 chart를 다시 그린다.                false로 지정하고 여러 설정 후에 [render](../classes/RcChartControl#render)를 호출해서 다시 그리게 할 수도 있다. |
