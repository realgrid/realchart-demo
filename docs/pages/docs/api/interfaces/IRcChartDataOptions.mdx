| field | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | N/A | row가 단일 값일 때 필드 이름.<br/> 이 필드 이름 속성을 갖는 json 객체로 저장된다.  기본값은 'y'. |
| fields | Array\<string> | N/A | 배열로 row 값들로 지정될 때, 각 항목에 해당되는 필드 이름들.<br/> 이 이름들을 속성으로 갖는 json 객체로 저장된다.  기본값은 ['x', 'y', 'z']; |
