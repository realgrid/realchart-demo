# RealChart Documentation

## 개발

### 설치

```sh
> cd docs
> yarn
```

### 디버깅

```sh
> yarn dev
```

### 빌드

```sh
> yarn build
```

### 실행

```sh
> yarn start
```

## 메뉴 구분

### 가이드

- mdx로 가이드 작성

### API 문서

- typedoc으로 문서 구조를 모델링한 다음 mdx로 렌더링

### 데모

- web에서 데모에 보여줄 목록을 demo.config.json파일로 정의
